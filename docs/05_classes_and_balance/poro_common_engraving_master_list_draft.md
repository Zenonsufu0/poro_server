# 포로 서버 공용각인 3슬롯 실제 목록표 초안

## 문서 목적
이 문서는 이미 정해진 공용각인 3슬롯 구조를 실제 목록 감각으로 정리한 초안이다.

핵심 목표:
- 회복/딜/디버프/생존 등 다양한 조합형 3슬롯 구조를 구체화한다.
- 각인이 세트, 룬, 잠재와 역할이 겹치지 않게 한다.
- 나중에 engraving seed로 옮기기 쉬운 이름과 효과 방향을 만든다.

---

## 1. 추천 목록

- 회복 강화
- 전투 집중
- 약점 추적
- 생존 본능
- 상태 확산
- 정밀 숙련
- 핵심 증폭
- 기동 전술
- 보호 강화
- 연계 숙련
- 자원 효율
- 역전의 의지

---

## 2. 효과 감각 예시

| 각인명 | 방향 | 레벨당 감각 |
|---|---|---|
| 회복 강화 | 회복 | 회복 효율 +2% |
| 전투 집중 | 딜 | 일반 피해 +1.5% |
| 약점 추적 | 조건부 딜 | 약점 노출 대상 피해 +2% |
| 생존 본능 | 생존 | 최대 체력 +2% |
| 상태 확산 | 디버프 | 상태이상 부여량 +2% |
| 정밀 숙련 | 정밀 | 정밀 태그 피해 +1.5% |
| 핵심 증폭 | 핵심 | 핵심 태그 피해 +1.5% |
| 기동 전술 | 이동 | 이동 후 다음 스킬 피해 +2% |
| 보호 강화 | 방어 | 보호막 효율 +2% |
| 연계 숙련 | 연계 | 연계 태그 피해 +1.5% |
| 자원 효율 | 운영 | 자원 획득량 +2% |
| 역전의 의지 | 위기 | 저체력 조건 피해/생존 소폭 증가 |

---

## 3. 한 줄 요약

**공용각인은 딜/생존/디버프/운영을 고르게 담은 3슬롯 조합형 구조로 두고, 레벨당 1.5~2% 정도의 완만한 상승으로 다양한 조합을 유도하는 것이 가장 좋다.**
