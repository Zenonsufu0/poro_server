# 포로 서버 직업각인 실제 목록표 초안

## 문서 목적
이 문서는 포로 서버 각 직업의 직업각인을 실제 목록 감각으로 정리한 초안이다.

핵심 목표:
- 전사 / 암살자 / 마법사의 플레이스타일 분기를 직업각인으로 고정한다.
- 공용각인과 달리, 직업각인은 플레이스타일을 강하게 바꾸는 핵심축으로 둔다.
- 나중에 class engraving seed로 옮기기 쉬운 이름과 효과 방향을 만든다.

---

## 1. 기본 철학

### 1-1. 직업각인은 강한 방향성을 줘야 한다
직업각인은 단순 수치 증가보다
- 자원 운용 변화
- 핵심 스킬 루프 변화
- 종결기 운용 변화
- 이동/생존/연계 구조 변화
를 주는 쪽이 좋다.

### 1-2. 무기마다 2갈래 정도의 감각이 좋다
너무 많으면 복잡하고,
너무 적으면 플레이스타일 차이가 약하다.

---

## 2. 전사 직업각인

### 대검
- `파쇄의 권위`
  - 균열 소모형
  - 패왕격/천붕 낙하 강화
  - 균열 5스택 종결 루프 강조

- `중갑의 폭주`
  - 철벽/중갑 운용형
  - 철벽 유지 브루저 또는 해제 후 폭딜 전환
  - 생존과 순간 화력의 이중 감각

### 태도
- `발도의 극의`
  - 강화 발도, 일섬, 심안베기 중심
  - 검세 운용 압축형
  - 납도/발도 감각 강화

- `잔심 연계`
  - 잔심 누적과 연계 루프 강화
  - 저장 일섬 선택 운용 가능
  - 고회전 연계형

### 창
- `관통의 이치`
  - 압박 보유/소모를 활용한 종결형
  - 관천격, 꿰뚫기 강화
  - 약점 노출 대상 압박 효율 상승

- `전선 기동`
  - 보폭 전환, 전선 돌파, 재배치 중심
  - 압박 보유형
  - 이동 후 연계/정밀 강화

---

## 3. 암살자 직업각인

### 단검
- `암영 집행`
  - 표식/후방/암영 소모형
  - 심장 가르기 강화
  - 짧은 순간 폭딜 극대화

- `은흔 추적`
  - 재배치/연계/표식 유지형
  - 그림자 횡단 루프 강화
  - 안정적 단일 압박

### 쌍검
- `혈월 연무`
  - 연계 점수 누적/폭딜형
  - 혈월 난무, 쌍월 절멸 강화
  - 손맛 좋은 회전형

- `잔광 추격`
  - 이동 후 타격, 추적, 잔광 마무리 중심
  - 고기동 지속 압박형

### 권갑
- `붕권 일격`
  - 압박 소모 종결형
  - 붕권 멸타 강화
  - 근접 폭딜형

- `연환 파쇄`
  - 연환권/금쇄 연타 고회전형
  - 압박 유지와 리듬형 근접 압박

---

## 4. 마법사 직업각인

### 스태프
- `폭뢰 술식`
  - 천뢰 붕괴, 번개 낙하 중심
  - 광역 순간폭딜 강화
  - 상태 대상 효율 상승

- `빙화 장막`
  - 화염 폭류/서리 감옥 중심
  - 장판/제어/상태 누적형

### 완드
- `공명 붕괴`
  - 공명 폭발, 차원 붕괴 강화
  - 단일/정밀 핵심기 중심

- `균열 추적`
  - 균열/약점 노출 관리형
  - 마도 봉쇄와 연계 중심
  - 보스전 정밀 압박형

### 룬
- `봉성 전개`
  - 룬진/조각 전개 후 종결
  - 봉성 붕락 강화
  - 준비형 폭딜 루프

- `결계 순환`
  - 짧은 전개/재배치/소모 회전형
  - 룬 격류, 결계 전환 중심
  - 고회전 설치형

---

## 5. 한 줄 요약

**직업각인은 전사는 자원 운용과 전환, 암살자는 표식/연계/순간 폭딜, 마법사는 공명/제어/전개 구조를 강하게 바꾸는 플레이스타일 분기축으로 두는 것이 가장 적합하다.**
