# 포로 서버 운영 경고 임계값 표 초안

## 문서 목적
이 문서는 운영 대시보드와 디스코드 봇, 관리자 경고 시스템에서 사용할 **1차 경고 임계값**을 정리한 초안이다.

주의:
- 이 값들은 운영 시작용 초안이다.
- 실제 시즌 운영과 알파/베타 통계 보고 조정해야 한다.
- 핵심은 무엇을 경고 대상으로 보고 어떤 수준에서 경고를 띄울지 기준을 만드는 것이다.

---

## 1. 경제 경고

| 항목 | 경고 조건 | 심각도 |
|---|---|---|
| 골드 순증 급등 | 최근 7일 평균 대비 **150% 이상** | WARNING |
| 골드 순증 폭증 | 최근 7일 평균 대비 **200% 이상** | CRITICAL |
| 특정 재화 공급 급감 | 최근 7일 평균 대비 **50% 이하** | WARNING |
| 특정 재료 과잉 공급 | 최근 7일 평균 대비 **180% 이상** | WARNING |
| 거래량 급증 | 최근 7일 평균 대비 **170% 이상** | WARNING |

---

## 6. 운영 메모

- WARNING은 운영자가 하루 안에 확인하는 수준
- CRITICAL은 메인 대시보드 최상단 고정 노출 수준
- 초기엔 경고를 너무 촘촘하게 잡지 말고, 자주 틀리는 항목은 나중에 조정한다.

---

## 7. 한 줄 요약

**운영 경고 임계값은 1차 기준으로 최근 7일 평균 대비 150~200% 이상 급등/급락, 보스 클리어율 15~20%p 이상 변화, 직업 DPS 편차 15% 이상, 생활 특정 자원 영지 비중 70% 이상 정도를 시작점으로 잡는 것이 무난하다.**
