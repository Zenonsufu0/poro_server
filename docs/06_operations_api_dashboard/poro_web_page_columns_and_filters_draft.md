# 포로 서버 웹 운영 페이지 컬럼 / 필터 표 초안

## 문서 목적
이 문서는 운영자용 웹 페이지에서 각 화면별로 어떤 컬럼과 필터를 제공하면 좋을지 정리한 초안이다.

핵심 목표:
- 운영자가 원하는 데이터를 빠르게 좁혀서 볼 수 있게 한다.
- 페이지별로 꼭 필요한 컬럼과 필터를 미리 정리한다.

---

## 1. 플레이어 상세 조회

### 컬럼
- 닉네임
- 직업
- 무기
- 총 강화 합
- 대표 잠재
- 최근 결전 클리어
- 최근 극상위 진척

### 필터
- 직업
- 무기
- 강화 구간
- 최근 접속일
- 보스 클리어 여부

---

## 2. 보스 통계

### 컬럼
- 보스명
- 시도 수
- 클리어율
- 평균 클리어 시간
- 평균 사망 수
- 위험 패턴

### 필터
- 보스 종류
- 결전 / 극상위
- 날짜 범위
- 파티 크기

---

## 3. 경제 통계

### 컬럼
- 품목명
- 평균 시세
- 거래량
- 공급량
- 전일 대비 변화
- 최근 7일 평균 대비 변화

### 필터
- 재료 / 소비품 / 보스 재료
- 날짜 범위
- 거래량 상위
- 급등/급락만 보기

---

## 4. 한 줄 요약

**웹 운영 페이지는 화면별로 핵심 컬럼과 좁히기 쉬운 필터를 미리 정의해두면 운영자가 원하는 데이터를 훨씬 빠르게 확인할 수 있다.**
