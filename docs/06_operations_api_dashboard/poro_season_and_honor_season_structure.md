# 포로 서버 시즌 / 명예 시즌 구조 초안

## 문서 목적
이 문서는 포로 서버가 나중에 시즌성 기록과 명예 경쟁을 운영할 경우의 구조를 정리한 초안이다.

핵심 목표:
- 시즌을 강한 숙제형이 아니라 기록형/명예형으로 운영한다.
- 서버 최초, 시즌 베스트, 시즌 한정 기록을 분리한다.
- 전승 보상과 시즌 기록이 충돌하지 않게 한다.

---

## 1. 기본 철학

### 영구 기록
- 서버 최초 클리어
- 최초 무사망
- 최초 전승 획득

### 시즌 기록
- 시즌 최단 클리어
- 시즌 최소 사망
- 시즌 최고 기여
- 시즌 특별 조건 달성

### 방향
시즌은 반복 숙제보다
**기록 경쟁과 명예 전시**에 더 가깝게 두는 편이 포로 서버답다.

---

## 2. 시즌 보상 추천

- 시즌 프레임
- 시즌 뱃지
- 시즌 칭호
- 시즌 기록 보관
- 소량 상징 재화

강한 전투 성능 보상은 비추천

---

## 3. 한 줄 요약

**시즌 구조는 숙제형 파밍 시즌보다 기록형/명예형 시즌에 가깝게 두고, 영구 최초 기록과 시즌별 베스트 기록을 분리하는 것이 가장 잘 맞는다.**
