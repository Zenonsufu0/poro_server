# 포로 서버 운영자 대시보드 MVP 범위 정리 초안

## MVP가 여기서 뜻하는 것
여기서 **MVP**는  
**“출시 전에 꼭 있어야 하는 최소 운영 대시보드 기능”**을 뜻한다.

즉:
- 있으면 좋은 것 전부를 다 만드는 게 아니라
- 운영자가 서버를 통제하는 데 **반드시 필요한 최소 범위만 먼저 구현**하는 것

쉽게 말하면,

**“이것만 있어도 운영이 가능하다”**  
라는 최소 대시보드 묶음이다.

---

## 왜 MVP를 따로 잡아야 하냐

지금까지 정리한 대시보드 구조는 꽤 넓다.

- 경제
- 성장
- 보스
- 전투 밸런스
- 명예의 전당
- 이상 징후
- 시즌 기록

이걸 한 번에 전부 다 만들면
- 시간 오래 걸리고
- 초반 구현량이 너무 커지고
- 실제 중요한 화면보다 부가 화면에 시간 쓰게 된다.

그래서 먼저

**출시 전 반드시 필요한 최소 운영 화면**
부터 확정하는 게 중요하다.

---

## MVP 판단 기준

아래 셋 중 하나에 해당하면 MVP에 넣는다.

1. **운영자가 매일 봐야 하는가**
2. **문제가 터졌을 때 바로 확인해야 하는가**
3. **밸런스 조정/경제 통제에 직접 필요한가**

반대로 아래는 MVP에서 뒤로 미뤄도 된다.

- 보기 좋지만 없어도 당장 운영 가능한 것
- 시즌 후반 명예 요소
- 장식성 높은 통계
- 심화 분석용 세부 화면

---

# 1. 운영자 대시보드 MVP 목표

출시 전 대시보드 MVP는 아래를 가능하게 해야 한다.

- 서버 경제가 이상한지 확인
- 결전/극상위 보스 난이도가 이상한지 확인
- 특정 직업/무기가 튀는지 확인
- 강화/큐브/재료 흐름이 과한지 확인
- 이상 징후가 생기면 바로 어디가 문제인지 좁히기

즉 한 줄로 하면:

**“운영자가 핵심 수치를 직접 보고 통제할 수 있게 만드는 최소 화면 묶음”**

---

# 2. MVP에 반드시 포함할 화면

## 2-1. 운영 메인 대시보드
이건 무조건 필요하다.

### 왜 필요하냐
운영자가 로그인하자마자
- 오늘 서버가 정상인지
- 경제/보스/직업 쪽에 문제 있는지
- 지금 제일 먼저 봐야 하는 경고가 뭔지
를 알아야 하기 때문이다.

### MVP 카드
- 전체 골드
- 오늘 골드 순증감
- 오늘 결전 클리어 수
- 오늘 극상위 도전 수
- 평균 보스 클리어율 변화
- 직업 평균 DPS 편차
- 가장 심각한 경고 1건

### MVP 그래프
- 최근 7일 골드 순증감
- 최근 7일 결전/극상위 도전 수
- 최근 7일 직업 평균 DPS 편차

---

## 2-2. 경제 대시보드
이것도 MVP에 포함하는 게 맞다.

### 왜 필요하냐
네 철학상
**운영자가 모르는 사이에 경제가 터지면 안 되기 때문**이다.

### MVP 카드
- 전체 골드
- 오늘 생성량
- 오늘 소멸량
- 오늘 순증감

### MVP 표/그래프
- 생성처 TOP 5
- 소모처 TOP 5
- 최근 7일 골드 생성/소멸 추이

---

## 2-3. 보스 대시보드
이건 핵심이다.

### 왜 필요하냐
포로 서버는 보스/던전 중심 서버라서
보스 난이도 이상을 못 보면 운영 자체가 안 된다.

### MVP 카드
- 전체 도전 수
- 전체 클리어 수
- 평균 클리어율
- 평균 클리어 시간
- 발악 진입률
- 발악 성공률

### MVP 표
보스별:
- 도전 수
- 클리어 수
- 클리어율
- 평균 클리어 시간
- 가장 위험한 패턴
- 1/2/3인 성공률

---

## 2-4. 전투 밸런스 대시보드
이것도 MVP 포함 추천이다.

### 왜 필요하냐
직업/무기 밸런스가 망가지면
보스 난이도와 경제까지 같이 흔들린다.

### MVP 카드
- 직업 평균 DPS 편차
- 무기 평균 DPS 편차
- 최고/최저 직업 격차

### MVP 표
- 직업별 평균 DPS
- 무기별 평균 DPS
- 직업별 생존률
- 직업별 평균 사망 수

---

## 2-5. 이상 징후 / 경고 대시보드
이건 가능하면 MVP에 꼭 넣는 게 좋다.

### 왜 필요하냐
운영자는 숫자 전체를 매번 다 뜯어보지 않는다.  
문제가 생기면 먼저 **경고를 보고 들어가야** 한다.

### MVP 경고 영역
- 골드 순증 급등
- 특정 보스 클리어율 급락
- 특정 직업 DPS 급등
- 특정 재료 공급량 급감
- 발악 성공률 0% 지속

### 표시 방식
- 심각도
- 경고 문구
- 발생 시각
- 상세 화면 링크

---

# 3. MVP에서 일단 제외해도 되는 것

이건 나중에 2차, 3차로 넣어도 된다.

## 3-1. 명예의 전당 전체 화면
이유:
- 중요하긴 하지만 출시 직전 즉시 운영 안정과는 우선순위가 다름
- 최소한 기록 저장만 먼저 하고, 전용 화면은 나중 가능

## 3-2. 시즌 상세 비교 화면
이유:
- 시즌이 실제로 굴러야 의미가 커짐
- 초반엔 일일/최근 7일 집계가 더 중요

## 3-3. 세트 / 잠재 / 외형 심화 통계 화면
이유:
- 필요하지만 출시 직전엔 경제/보스/직업이 더 우선
- 심화 분석은 후속 추가 가능

---

# 4. 출시 전 꼭 구현해야 하는 MVP 카드 목록

## 메인
- 전체 골드
- 오늘 골드 순증감
- 오늘 결전 클리어 수
- 오늘 극상위 도전 수
- 평균 클리어율 변화
- 직업 DPS 편차
- 최상위 경고 1건

## 경제
- 생성량
- 소멸량
- 순증감
- 생성처 TOP 5
- 소모처 TOP 5

## 보스
- 보스별 도전 수
- 보스별 클리어율
- 보스별 평균 시간
- 보스별 위험 패턴
- 파티 규모별 성공률

## 전투 밸런스
- 직업별 평균 DPS
- 무기별 평균 DPS
- 생존률
- 편차 경고

## 경고
- 경제 경고
- 보스 경고
- 직업 경고
- 재료 경고

---

# 5. 출시 후 2차로 추가할 것

## 5-1. 성장 심화 화면
- 강화 단계 히스토그램
- 잠재 분포
- 세트 제작 분포
- 유니크 제작 통계

## 5-2. 명예 / 시즌 기록 전용 화면
- 최초 클리어 기록
- 무사망 기록
- 솔플 기록
- 극상위 미클리어 현황

## 5-3. 심화 재료 흐름 화면
- 재료별 유통량
- 공급/소비 상세
- 테마별 재료 병목

---

# 6. 출시 후 3차로 추가할 것

## 6-1. 고급 비교 화면
- 시즌 간 비교
- 패치 전후 비교
- 클래스 메타 변화 비교

## 6-2. 심화 분석 화면
- 패턴별 사망 히트맵
- 페이즈별 포기율
- 무력화 성공/실패 상세

## 6-3. 명예/치장 컬렉션 화면
- 극상위 칭호 보유 현황
- 외형 보유 분포
- 시즌 업적 달성률

---

# 7. 현실적인 MVP 최종 범위

## 꼭 만든다
- 운영 메인
- 경제
- 보스
- 전투 밸런스
- 경고

## 저장은 해두고 화면은 나중
- 명예의 전당
- 시즌 기록
- 성장 심화
- 재료 심화

이렇게 가는 게 제일 현실적이다.

---

# 8. MVP 화면 흐름 추천

## 운영자 실제 사용 흐름
1. 메인 대시보드 확인
2. 경고 카드 확인
3. 경제 / 보스 / 전투 밸런스 중 이상 영역 클릭
4. 상세 표 확인
5. 필요 시 관리자 명령/디코 요약으로 교차 확인
6. 밸런스 조정 표 참고 후 수정 여부 결정

이 흐름만 잘 돌아도 운영 통제력은 꽤 높다.

# 9. 한 줄 요약

### MVP 의미
**출시 전에 반드시 있어야 하는 최소 운영 대시보드 범위**

### 이 문서 기준 MVP
**메인 / 경제 / 보스 / 전투 밸런스 / 경고 화면까지는 출시 전에 꼭 구현하고, 명예의 전당·성장 심화·시즌 심화는 2차 이후로 넘긴다.**
