# 포로 서버 데스카운트 / 음식 / 만찬 적용 규칙 초안

## 문서 목적
이 문서는 포로 서버 보스전의 **데스카운트 규칙**과 생활 시스템의 **음식 / 만찬 적용 규칙**을 정리한 초안이다.

핵심 목표:
- 보스전 실패 조건을 명확히 한다.
- 파티 플레이 긴장감과 학습 여지를 동시에 확보한다.
- 음식과 만찬의 중첩 규칙을 단순하고 직관적으로 고정한다.

---

## 1. 보스 데스카운트 규칙

### 1-1. 기본 구조
보스전은 **파티 공용 데스카운트**를 사용한다.

### 1-2. 인원수별 데스카운트
- 1인: **3 데카**
- 2인: **4 데카**
- 3인: **5 데카**

### 1-3. 사망 시 처리
- 사망 시 공용 데스카운트 1 소모
- 데스카운트가 남아 있으면 부활 가능
- 데스카운트가 0이면 더 이상 부활 불가

### 1-4. 실패 조건
- 데스카운트가 0인 상태에서
- 파티 전원이 사망하면
- 해당 보스전은 실패 처리

즉:
- 데카가 남아 있으면 복귀 가능
- 데카가 0이어도 생존자가 있으면 계속 진행 가능
- 데카가 0인데 모두 죽으면 종료

---

## 2. 데스카운트 구조의 의도

### 장점
- 죽음에 분명한 긴장감이 생김
- 실수 1~2회는 학습 기회로 허용 가능
- 마인크래프트 전투 환경에서 억울함을 조금 줄일 수 있음
- 파티 생존 관리가 중요해짐

### 운영 의도
- 로스트아크식 “죽으면 끝”보다 덜 가혹하게
- 메이플식 데카 구조를 참고하되
- 파티 공용 데카로 협동 압박을 강화

---

## 3. 음식 / 만찬 적용 규칙

## 3-1. 기본 구조
- **음식은 개인 적용**
- **만찬은 파티 전체 적용**

---

## 3-2. 사용 가능 위치
### 만찬
- **보스방에서 사용 가능**
- **영지에서 사용 가능**

### 음식
- 개인 사용
- 일반 필드 / 영지 / 보스 준비 상황에서 사용 가능

---

## 3-3. 중첩 규칙
### 최상위 원칙
- **음식과 만찬은 동시에 적용되지 않는다**
- **둘 중 하나만 적용 가능하다**

즉:
- 음식 버프 활성 상태에서 만찬 먹으면 → 음식 제거, 만찬 적용
- 만찬 버프 활성 상태에서 음식 먹으면 → 만찬 제거, 음식 적용

---

## 3-4. 같은 카테고리 내 중첩 규칙
### 음식
- 음식은 **한 번에 하나만 적용**
- 다른 음식 먹으면 기존 음식 효과 제거
- 새 음식 효과로 대체
- 지속시간은 새 효과 기준으로 초기화

### 만찬
- 만찬도 **한 번에 하나만 적용**
- 다른 만찬 먹으면 기존 만찬 제거
- 새 만찬 효과로 대체
- 지속시간은 새 효과 기준으로 초기화

---

## 3-5. 적용 예시
### 예시 1
- 회복 음식 적용 중
- 설원의 만찬 사용
- 결과: 회복 음식 제거, 설원의 만찬 적용

### 예시 2
- 화상 저항 음식 적용 중
- 다른 음식 사용
- 결과: 기존 음식 제거, 새 음식 적용, 지속시간 초기화

### 예시 3
- 제국 만찬 적용 중
- 정령 숲 만찬 사용
- 결과: 기존 만찬 제거, 새 만찬 적용, 지속시간 초기화

---

## 4. 규칙의 의도

### 음식
- 개인 준비용
- 간단하고 직관적인 선택
- 여러 버프를 억지로 겹쳐 먹는 숙제 방지

### 만찬
- 파티 준비용
- 상징성 있는 생활 고급 산출물
- 파티 전체 준비의 느낌 제공

### 공통 의도
- 준비는 의미 있게
- 중첩 숙제는 최소화
- 버프 관리 규칙은 직관적으로

---

## 5. 구현용 규칙 요약

### 데스카운트
- `party_size = 1 -> death_count = 3`
- `party_size = 2 -> death_count = 4`
- `party_size = 3 -> death_count = 5`
- 사망 시 1 차감
- 남은 데카 > 0 이면 부활 가능
- 남은 데카 = 0 이면 부활 불가
- 남은 데카 = 0 상태에서 전원 사망 시 실패

### 버프 규칙
- 개인 음식 버프 슬롯: 1개
- 파티 만찬 버프 슬롯: 1개처럼 보이지만 실제 적용은 개인 상태에 들어감
- 개인은 최종적으로 **음식/만찬 통합 슬롯 1개**만 유지
- 새 음식/만찬 적용 시 기존 버프 제거 후 새 버프 등록
- 지속시간은 새 버프 기준으로 초기화

---

## 6. 한 줄 요약

**보스전은 1인 3 / 2인 4 / 3인 5의 파티 공용 데스카운트를 사용하고, 데카가 남아 있으면 부활 가능하며 데카가 0인 상태에서 전원 사망하면 실패한다. 음식은 개인 적용, 만찬은 파티 전체 적용이지만 최종적으로 음식/만찬은 통합 1슬롯만 사용하며 새 음식을 먹거나 새 만찬을 먹으면 기존 효과는 제거되고 지속시간이 초기화된 새 효과로 대체된다.**
