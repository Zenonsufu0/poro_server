# 포로 서버 보스 보상 드랍 테이블 상세 초안

## 문서 목적
이 문서는 포로 서버 보스 보상의 **실제 드랍 테이블 형태**를 정리한 초안이다.

핵심 목표:
- 결전 보스와 극상위 보스의 드랍 구조를 실제 지급 테이블처럼 정리한다.
- 개인 기본보상 / 추가보상 / 최초 클리어 보상 / 명예 보상을 분리한다.
- Codex가 보상 resolver / distributor / reward config seed를 만들 수 있게 한다.

주의:
- 수치는 최종 확정본이 아니라 1차 기준표다.
- 운영/테스트 단계에서 조정 가능하다.

---

## 1. 결전 보스 드랍 구조

### 기본 지급
모든 결전 보스는 개인 기준 아래를 기본 지급
- 골드
- 강화석
- 큐브
- 결전 재료
- 테마 재료

### 추가 지급
- 희귀 테마 재료
- 생활/가공 보조 재료
- 유니크 제작 보조 재료(아주 낮은 확률)

## 3. 극상위 보스 드랍 구조

### 기본 지급
- 극상위 증표 x1
- 상징 재화 1~3
- 소량 골드 2000~6000
- 기록 처리

### 최초 클리어 추가
- 칭호
- 업적
- 치장 보상
- 명예의 전당 등록

### 반복 클리어
- 소량 상징 재화
- 소량 골드
- 낮은 확률 전시/치장 재료(선택)

## 5. 드랍 구조 구현 메모

- 결전은 reward config table로 고정량 + 확률 추가보상을 같이 처리
- 극상위는 최초 클리어 분기 로직이 별도 필요
- 명예 보상은 inventory 지급보다 record/achievement service와 연결하는 편이 안정적

---

## 6. 한 줄 요약

**결전 보스는 고정 기본보상 + 낮은 확률 추가보상 구조, 극상위는 증표/상징 재화/최초 클리어 명예 보상 구조로 두는 것이 가장 안정적이며, 이 표를 그대로 reward resolver seed의 1차 기준으로 사용할 수 있다.**
