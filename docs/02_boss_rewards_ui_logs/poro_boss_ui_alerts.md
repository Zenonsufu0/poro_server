# 포로 서버 결전 / 극상위 UI·알림 구조 초안

## 문서 목적
이 문서는 포로 서버 결전 보스 / 극상위 보스에서 필요한 UI, 알림, 경고, 연출 표시 구조를 정리한 초안이다.

핵심 목표는 다음과 같다.

- 유저가 **지금 무슨 규칙이 적용 중인지 즉시 이해**할 수 있어야 한다.
- 억울한 죽음보다 **배우고 대응하는 보스전**이 되어야 한다.
- 결전/극상위의 복잡한 기믹을 **마인크래프트 환경에서 읽기 쉽게 시각화**해야 한다.

---

## 1. 기본 철학

### 1-1. 표시 우선순위
보스전 UI는 아래 순서로 중요하다.

1. **지금 당장 죽을 수 있는 정보**
2. **곧 처리해야 하는 개인 기믹**
3. **파티 전체가 알아야 하는 전장 규칙**
4. **딜타임 / 무력화 기회**
5. **기록 / 연출 / 장식 정보**

즉 예쁘게 보이는 것보다  
**지금 당장 무엇을 해야 하는지**가 먼저 보이게 해야 한다.

---

### 1-2. 한 화면에 너무 많은 정보를 주지 않는다
결전/극상위는 정보량이 많지만,  
한 번에 너무 많은 텍스트를 띄우면 오히려 읽히지 않는다.

추천 원칙:
- **즉시 행동 요구**는 액션바/타이틀
- **개인 상태**는 보스바 아래 또는 사이드 상태창
- **전장 규칙**은 보스 패턴 시작 시 짧은 문구 + 시각 연출
- **기록/업적**은 전투 종료 후 별도 표시

---

### 1-3. 같은 종류의 정보는 같은 방식으로 보여준다
예를 들면:
- 표식류 = 머리 위 아이콘 + 액션바 문구
- 전장 위험 = 바닥 파티클/블록/색 변화
- 무력화 = 타이틀 + 보스바 강조
- 발악 패턴 = 타이틀 + 사운드 + 보스바 색 변화

이렇게 **표시 문법을 통일**하는 게 중요하다.

---

## 2. 추천 UI 채널 분리

### 2-1. 타이틀 / 서브타이틀
사용 용도:
- 페이즈 전환
- 즉시 대응이 필요한 대형 패턴
- 발악 패턴 시작
- 가짜 클리어 후 진실 페이즈 진입

예:
- `설혼 안개`
- `과열 단계 상승`
- `사형 집행 선언`
- `진실 페이즈 돌입`

추천 규칙:
- 한 번 뜨면 **짧고 강하게**
- 긴 설명문 금지
- 1~2줄 이내

---

### 2-2. 액션바
사용 용도:
- 지금 당장 개인이 해야 할 행동
- 짧은 카운트다운
- 표식 지속시간
- 위험 경고

예:
- `백빙 낙인: 3.2초 후 폭발`
- `침입 죄목 적용 중`
- `안전지대로 이동하라`
- `심핵 붕괴 임계: 8초`

추천 규칙:
- 결전/극상위 핵심 정보는 액션바를 자주 활용
- 짧고 반복적으로 보여줄 것
- 숫자 카운트다운은 액션바가 제일 좋음

---

### 2-3. 보스바
사용 용도:
- 보스 체력
- 페이즈 상태
- 특수 패턴 상태

추천 확장:
- 보스바 이름 변경
- 보스바 색 변경
- 보스바 분할 수 변경

예:
- 평시: 빨강
- 무력화 가능 상태: 노랑
- 발악 패턴: 보라/흰색
- 가짜 클리어 후 진실 페이즈: 검정/금색

추천 문구 예시:
- `황혼의 대재판관 세르카인 - 공판 개시`
- `백빙룡 엘드하임 - 백빙 폭설`
- `천공사수 세트라 - 진실 공간`

---

### 2-4. 사이드 상태창 / 보조 HUD
사용 용도:
- 스택형 디버프
- 죄목
- 침식 게이지
- 판결 스택
- 고요 침식
- 과열 단계
- 냉기 침식

예:
- `판결 스택: 3`
- `고요 침식: 42%`
- `과열 단계: 2`
- `냉기 침식: 4중첩`

추천 규칙:
- 장기적으로 누적되는 정보는 여기로
- 전투 내내 유지되는 개인 상태에 적합

---

### 2-5. 바닥 연출 / 파티클 / 블록 변화
사용 용도:
- 실제 위험 구역
- 안전지대
- 구획 분할
- 분화선
- 가짜 안전지대
- 반사 광선 경로
- 생장 역류 구역

이 부분은 결전/극상위에서 사실상 가장 중요하다.

추천 규칙:
- **위험 = 명확한 색**
- **안전 = 명확한 대비**
- **가짜 안전지대 = 미세한 차이**
- **테마별 색 규칙 유지**

---

### 2-6. 사운드
사용 용도:
- 전용 기믹 발동
- 발악 패턴 진입
- 무력화 성공/실패
- 표식 폭발 직전

추천 규칙:
- 사운드는 “정보 보조”로 사용
- 너무 남발하지 않기
- 전용 기믹마다 대표음 1개씩 두면 좋음

---

## 3. 결전/극상위에서 반드시 표시해야 하는 정보

### 3-1. 개인 표식류
반드시 보여야 한다.
- 낙인
- 선고
- 그림자 도약 표식
- 과열 표식
- 공명 표식

추천 표시 방식:
- 액션바
- 머리 위 아이콘
- 몸 주변 파티클
- 지속시간 숫자 표시 가능하면 좋음

---

### 3-2. 전장 안전/위험 구역
반드시 명확해야 한다.
- 안전지대
- 구획 분할
- 폭풍 장벽
- 혹한 외곽
- 과열 지대
- 생장 역류
- 반사 광선 경로

추천 표시 방식:
- 바닥 색 차이
- 파티클 테두리
- 위험 지역의 사운드/잔향
- 안전 구역의 밝기/문양 차이

---

### 3-3. 무력화 / 딜타임
이건 유저가 놓치면 억울하다.

추천 표시 방식:
- 타이틀: `심핵 노출`
- 보스바 색 변화
- 액션바: `지금 집중 공격`
- 사운드: 짧은 성공음

---

### 3-4. 발악 패턴
무조건 크게 보여야 한다.

추천 표시 방식:
- 타이틀
- 보스바 색/이름 변경
- 액션바 카운트다운
- 전장 전체 사운드
- 화면 연출(가능 범위 내)

예:
- `최종 판결`
- `심핵 폭주`
- `백빙 심핵 최종 노출`
- `폭풍핵 폭주`

---

### 3-5. 가짜 클리어 / 진실 페이즈
사하르 상인연합 극상위는 이게 핵심이다.

추천 표시 방식:
- 0%에서 잠깐 승리처럼 보이게
- 보스바 제거
- 짧은 정적
- 이후:
  - 화면 어두워짐
  - 타이틀: `진실은 아직 끝나지 않았다`
  - 새로운 보스바 등장
  - 공간 전환 연출

첫 클리어 때는 충격을 살리고,
반복 클리어 때는 일부 연출 축약 가능하도록 설계 추천.

---

## 4. 테마별 UI 연출 방향

### 4-1. 수도권
핵심:
- 판결
- 법정
- 금지/허용
- 죄목

추천 표현:
- 금색 / 흑금색
- 선고 문양
- 구획선 명확 표시
- 죄목 아이콘

특수 UI:
- `침입`, `불복종`, `도주` 같은 죄목이
  개인 HUD에 작게 떠야 함
- 판결 스택은 누적 숫자가 잘 보여야 함

---

### 4-2. 서부 황야
핵심:
- 생존
- 추적
- 폭풍
- 붕괴

추천 표현:
- 갈색 / 적갈색 / 황토색
- 시야 흐림
- 모래 파티클
- 바닥 붕괴 문양

특수 UI:
- 폭풍 장벽은 바닥보다 **전방 시야 전체 압박감**이 중요
- 안전지대가 잠깐뿐인 느낌을 시각적으로 줘야 함

---

### 4-3. 동부 숲
핵심:
- 안개
- 분신
- 뿌리
- 생장

추천 표현:
- 청록 / 녹색 / 보라 안개
- 뿌리 문양
- 실루엣형 분신
- 숲의 울림 같은 사운드

특수 UI:
- 본체 판별 힌트는 너무 명확하면 재미없고,
  너무 숨기면 억까가 된다.
- “살짝 보이는 차이”를 주는 연출이 중요

---

### 4-4. 북부 설원
핵심:
- 설혼 안개
- 빙결
- 백빙 낙인
- 폭설

추천 표현:
- 흰색 / 푸른색 / 은빛
- 시야 감소
- 서리 파티클
- 냉기 HUD 효과

특수 UI:
- 냉기 침식, 낙인, 혹한 외곽은
  누적/시간이 매우 잘 보여야 한다.
- 너무 안 보이면 억울함이 커짐

---

### 4-5. 남부 화산
핵심:
- 낙인
- 분화선
- 과열
- 폭발

추천 표현:
- 붉은색 / 주황색 / 검은 잿빛
- 균열선
- 불씨
- 열기 왜곡 효과

특수 UI:
- 분화선은 “곧 터질 선”이라는 걸 직관적으로 보여줘야 한다.
- 과열 단계는 HUD 또는 보스바로 계속 인식 가능해야 좋다.

---

### 4-6. 사하르 상인연합
핵심:
- 기만
- 환영
- 가짜 정답
- 태양 반사

추천 표현:
- 금빛 / 모래빛 / 청색 보석광
- 허상 이펙트
- 빛 반사
- 가짜 안전지대

특수 UI:
- 가짜/진짜 차이는 아주 미세해야 하지만,
  완전 랜덤처럼 느껴지면 안 된다.
- 첫 클리어의 가짜 승리 연출은 이 테마의 핵심 자산이다.

---

### 4-7. 아르카논
핵심:
- 레이저
- 공명
- 시스템
- 구획 재배치

추천 표현:
- 푸른색 / 보라색 / 백색 기계광
- 격자 바닥
- 레이저선
- 공명 노드

특수 UI:
- 구획이 바뀔 때 전장 변화가 명확해야 함
- 안전 구역 표시가 너무 복잡하면 읽기 힘들 수 있음
- 규칙적인 기계 연출이 중요

---

## 5. 표시 문법 통일안

### 위험 구역
- 붉은색 / 테마색 짙은 경고
- 굵은 바닥 문양
- 강한 파티클

### 안전 구역
- 밝은색 / 대비 문양
- 맥동 또는 잔잔한 빛

### 가짜 안전 구역
- 안전 구역과 비슷하지만
- 문양이 약간 틀리거나
- 깜빡임 / 왜곡 / 지연된 변화로 차별

### 개인 표식
- 머리 위 아이콘
- 액션바 카운트다운
- 몸 주변 파티클

### 장기 누적 상태
- 사이드 HUD / 상태창
- 숫자 또는 게이지 표시

### 무력화
- 타이틀 + 보스바 색 변화 + 성공 사운드

### 발악
- 타이틀 + 보스바 변화 + 액션바 카운트다운 + 전장 전체 연출

## 6. 꼭 필요한 UI 요소 목록

### 필수
- 보스바
- 액션바
- 타이틀/서브타이틀
- 바닥 경고 연출
- 개인 표식 연출
- 장기 누적 상태 표시

### 있으면 좋은 것
- 간단한 사이드 상태 UI
- 상단 페이즈 텍스트
- 무력화 성공/실패 아이콘
- 첫 클리어 전용 연출

---

## 7. 억울함 줄이기 원칙

### 반드시 지켜야 할 것
- 즉사급 패턴은 반드시 선행 경고가 있어야 함
- 개인 책임 패턴은 누가 대상인지 명확해야 함
- 안전지대는 최소한 “읽을 수는 있어야” 함
- 가짜 기믹도 완전 랜덤처럼 느껴지면 안 됨
- 발악 패턴은 실패 이유가 분명해야 함
