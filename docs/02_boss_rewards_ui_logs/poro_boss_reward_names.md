# 포로 서버 결전 / 극상위 보상 명칭 초안

## 문서 목적
이 문서는 포로 서버의 결전 보스 / 극상위 보스 보상에 사용될 **명칭 체계**를 정리한 초안이다.

정리 대상:
- 칭호
- 업적명
- 상징 재화
- 외형/장식 이름
- 파티클/오라 이름

---

## 1. 명칭 설계 원칙

### 1-1. 결전 보스
결전 보스 보상 명칭은
- 너무 거창한 “최종 정복자” 느낌보다
- 해당 테마를 돌파하고 도전 자격을 얻었다는 느낌
으로 잡는다.

즉:
- 실전 파밍 단계
- 후반 콘텐츠 돌파
- 숙련의 시작점

---

### 1-2. 극상위 보스
극상위 보상 명칭은
- 명예
- 상징성
- 서버 대표성
- 업적감

이 강해야 한다.

즉:
- 다시 못 얻어도 이상하지 않은 이름
- 보스를 클리어한 사실 자체를 드러내는 이름
- 과시성 있는 이름

---

## 9. 결전 / 극상위 명칭 톤 차이 정리

### 결전 보스 명칭 톤
- 돌파
- 생환
- 시련
- 제압
- 파쇄
- 차단

즉:
**“넘어섰다 / 돌파했다 / 준비를 마쳤다”** 느낌

### 극상위 보스 명칭 톤
- 정복
- 종결
- 끝
- 생존자
- 진실을 본 자
- 넘은 자
- 마지막 숨결

즉:
**“증명했다 / 살아남았다 / 이겼다 / 기억될 자격이 있다”** 느낌

## 10. 추천 우선 확정 항목

먼저 확정하면 좋은 것:
1. 극상위 칭호
2. 극상위 업적명
3. 상징 재화 이름
4. 결전 재화 이름
5. 치장/오라 이름

이 순서가 좋다.

이유:
- 극상위 쪽이 상징성이 강해서 먼저 정하면 전체 톤이 쉽게 고정됨
- 이후 결전 쪽 명칭은 그보다 한 단계 아래 위상으로 맞추기 쉬움
