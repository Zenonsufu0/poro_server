# 포로 서버 결전 / 극상위 보상 상세안 초안

## 문서 목적
이 문서는 포로 서버의 결전 보스 / 극상위 보스 보상 철학을 실제 운영 기준으로 세분화한 초안이다.

---

## 1. 핵심 철학

### 결전 보스
결전 보스는 **후반 성장의 핵심 파밍 보스**다.

- 극상위 보스 도전을 위한 준비 단계
- 반복 클리어 가치가 높아야 함
- 골드, 강화석, 큐브, 심층 전용 재료, 상위 보조 재료를 안정적으로 줘야 함
- 성장 체감이 분명해야 함

한 줄 요약:

**결전 보스는 성장의 완성선을 위한 실전 파밍 보스다.**

---

### 극상위 보스
극상위 보스는 **업적형 최종 도전 콘텐츠**다.

- 필수 성장 루트가 아님
- 세트/일반 종결 필수 재료를 주지 않음
- 못 깨도 시즌 전체 구조가 안 막혀야 함
- 클리어 자체가 가치가 되어야 함
- 명예, 기록, 칭호, 외형, 치장 보상이 중심

한 줄 요약:

**극상위 보스는 성장보다 증명과 명예를 주는 최종 도전 콘텐츠다.**

---

## 2. 결전 보스 보상 구조

### 반복 클리어 기본 보상
- 골드: 높음
- 강화석: 높음
- 큐브: 높음
- 심층 전용 재료: 높음
- 상위 보조 재료: 중~높음

### 결전 보스 보상 역할
- 후반부 실질 성장 보스
- 극상위 준비용 핵심 보스
- 시즌 후반 기준 반복 파밍 핵심 루트
- 일반 유저 상위권 / 숙련 유저 종결선 연결

### 결전 보스가 줘야 하는 것
- 골드
- 강화석
- 큐브
- 심층 전용 재료
- 상위 전승/제작 보조 재료
- 낮은 확률의 치장성 보상
- 업적 조건
- 테마별 희귀 제작 재료 일부

### 결전 보스가 주면 안 되는 것
- 극상위 보상급 상징 칭호
- 서버 명예의 전당 기록급 보상
- 극상위 전용 과시성 외형

---

## 3. 극상위 보상 구조

### 1회 클리어 보상
- 테마 전용 칭호
- 업적 해금
- 서버 기록 등록
- 특수 파티클 / 프로필 장식
- 테마 상징 외형 1종
- 테마 상징 재화 소량

### 첫 서버 클리어 보상
- 명예의 전당 등록
- 해당 테마 거점/NPC 대사 변화 가능
- 기념비 문구 / 서버 기록 저장
- 시즌 한정 표시

### 반복 클리어 보상
- 상징 재화 소량
- 매우 낮은 확률의 치장 보상
- 컬렉션형 보상
- 자기만족형 기록 갱신

### 극상위 보상이 주면 안 되는 것
- 일반 유저 종결 필수 재료
- 세트 흔적 핵심 파밍 루트
- 안 깨면 큰 손해가 되는 성장 아이템
- 극상위 클리어 강제가 되는 핵심 능력치 보상

---

## 4. 결전 / 극상위 차이 표

| 구분 | 결전 보스 | 극상위 보스 |
|---|---|---|
| 성격 | 후반 파밍 핵심 | 업적형 최종 도전 |
| 반복 파밍 가치 | 높음 | 낮음 또는 제한적 |
| 성장 필수성 | 있음 | 없음 |
| 주요 보상 | 골드, 강화석, 큐브, 재료 | 칭호, 업적, 외형, 기록 |
| 못 깼을 때 손해 | 어느 정도 있음 | 거의 없어야 함 |
| 도전 이유 | 성장 | 명예, 증명 |

## 5. 결전 보스 보상 템플릿

### 기본 보상 묶음
- 골드 다량
- 강화석 다량
- 큐브 다량
- 심층 전용 재료
- 상위 보조 재료

### 저확률 보상
- 외형 조각
- 테마 장식 아이템
- 희귀 문양
- 업적용 수집품

### 1회 업적 보상
- 해당 결전 보스 처치 업적
- 소형 칭호 또는 업적 포인트
- 테마 내 다음 콘텐츠 해금 표시

---

## 6. 극상위 보상 템플릿

### 1회 확정 보상
- 테마 전용 칭호
- 업적
- 클리어 기록
- 프로필 뱃지
- 테마 전용 파티클 또는 장식
- 상징 재화

### 반복 보상
- 극소량 상징 재화
- 아주 낮은 확률 치장 보상
- 기록 갱신 목적

### 첫 서버 클리어 특별 보상
- 서버 기록
- 시즌 명예 표시
- 기념비/동상/명예의 전당 연계 가능

## 7. 테마별 결전 보상 방향

### 수도권 결전 - 황혼의 대재판관 세르카인
- 보상 느낌: 제국 재판 기록, 법정/심판 분위기
- 추천 보상 키워드:
  - 심판 문양
  - 금빛 인장
  - 판결 기록서
  - 제국 재판 조각

### 서부 결전 - 사막패왕 하자드
- 보상 느낌: 생존, 모래폭풍, 황야 정복
- 추천 보상 키워드:
  - 황야 송곳니
  - 폭풍 모래핵
  - 붉은 협곡 파편
  - 생존자의 문장

### 동부 결전 - 별그늘 여왕 카실레아
- 보상 느낌: 안개, 그림자, 정령, 숲의 흔적
- 추천 보상 키워드:
  - 별그늘 꽃잎
  - 안개 결정
  - 정령 뿌리핵
  - 숲의 표식

### 북부 결전 - 봉인의 백빙 사제 모르바인
- 보상 느낌: 설혼, 봉인, 빙결 의식
- 추천 보상 키워드:
  - 백빙 의식편
  - 설혼 결정
  - 봉인의 조각
  - 혹한 문양

### 남부 결전 - 심홍화신 라그네스
- 보상 느낌: 홍염, 낙인, 분화
- 추천 보상 키워드:
  - 심홍 낙인핵
  - 분화 결정
  - 화맥 조각
  - 홍염의 문장

### 사하르 결전 - 미라지 군주 아자켈
- 보상 느낌: 기만, 환영, 금고, 계약
- 추천 보상 키워드:
  - 환영 주화
  - 금고 파편
  - 계약 인장
  - 신기루 수정

### 아르카논 결전 - 심핵 통제자 아르케논
- 보상 느낌: 공명, 심핵, 통제 시스템
- 추천 보상 키워드:
  - 통제 회로
  - 심핵 잔광
  - 공명 코어 조각
  - 과충전 도면

## 8. 테마별 극상위 보상 방향

### 수도권 극상위 - 제국 심연집행자 아우렐
- 칭호 방향:
  - 황도의 심판자
  - 최후 집행 생존자
  - 유죄를 넘어선 자
- 치장 방향:
  - 흑금 파티클
  - 처형 검 문양
  - 제국 집행 인장
- 상징 재화:
  - 제국 인장

### 서부 극상위 - 모래폭군 네브카
- 칭호 방향:
  - 황야 생존자
  - 폭풍 너머의 자
  - 매몰을 거부한 자
- 치장 방향:
  - 사막 폭풍 파티클
  - 붉은 모래 오라
  - 황야 패왕 문양
- 상징 재화:
  - 폭풍 잔핵

### 동부 극상위 - 고요의 심수목 엘테론
- 칭호 방향:
  - 고요를 들은 자
  - 심수목의 응답자
  - 생장의 끝에 선 자
- 치장 방향:
  - 나뭇잎/정령 파티클
  - 세계수 뿌리 문양
  - 청록 숲의 오라
- 상징 재화:
  - 심수목 잎맥

### 북부 극상위 - 백빙룡 엘드하임
- 칭호 방향:
  - 백빙을 넘은 자
  - 혹한의 생존자
  - 침묵의 설원 정복자
- 치장 방향:
  - 서리 파티클
  - 빙룡 문양
  - 백빙 숨결 오라
- 상징 재화:
  - 백빙 조각

### 남부 극상위 - 불멸화룡 아그네르
- 칭호 방향:
  - 종화를 넘은 자
  - 불멸의 불꽃 생존자
  - 홍염 정복자
- 치장 방향:
  - 화염 파티클
  - 용심 문양
  - 홍염 잔불 오라
- 상징 재화:
  - 홍염 심편

### 사하르 극상위 - 천공사수 세트라
- 칭호 방향:
  - 진실을 본 자
  - 허상을 꿰뚫은 자
  - 천공의 거짓을 부순 자
- 치장 방향:
  - 금빛 반사 파티클
  - 사막 허상 오라
  - 태양 조각 문양
- 상징 재화:
  - 천공 각편

### 아르카논 극상위 - 심연룡 카르멘
- 칭호 방향:
  - 심핵 차단자
  - 공명 종결자
  - 차원 붕괴를 넘은 자
- 치장 방향:
  - 공명 파티클
  - 차원 균열 오라
  - 심핵 문양
- 상징 재화:
  - 심핵 파편

- 동부 숲: **고요의 심장 파열**
- 북부 설원: **백빙의 끝**
- 남부 화산: **종화 돌파**
- 사하르 상인연합: **거짓 너머의 진실**
- 아르카논: **공명 종언**

## 10. 서버 기록 방식 초안

### 저장 추천 항목
- 최초 클리어 날짜
- 최초 클리어 파티 이름/유저
- 클리어 시도 횟수
- 클리어 시간
- 무사망 여부
- 시즌 내 최초 클리어 여부

### 표시 추천 위치
- 거점 명예의 전당 NPC
- 웹/디스코드 봇 연동
- 시즌 기록 페이지
- 보스 입장 전 안내 UI

---

## 11. 첫 클리어 / 반복 클리어 분리 원칙

### 결전 보스
- 첫 클리어: 업적 + 소형 추가 보상
- 반복 클리어: 실질 성장 보상 핵심

### 극상위 보스
- 첫 클리어: 핵심 명예 보상 대부분 지급
- 반복 클리어: 상징 재화 + 낮은 확률 치장
- 반복 클리어의 성장 효율은 낮아야 함

---

## 12. 절대 지켜야 할 정책

### 결전 보스
- 잡으면 성장한다.

### 극상위 보스
- 잡으면 증명된다.

이 문장을 기준으로 계속 판단하면 보상 설계가 흔들리지 않는다.
