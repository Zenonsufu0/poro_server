# 포로 서버 테마별 UI 문구 모음 초안

## 문서 목적
이 문서는 포로 서버 결전 보스 / 극상위 보스에서 사용할 **UI 문구 톤과 예시 문장**을 정리한 초안이다.

정리 대상:
- 페이즈 전환 문구
- 전용 기믹 발동 문구
- 개인 경고 문구
- 무력화 성공 / 실패 문구
- 발악 패턴 문구
- 첫 클리어 / 서버 최초 클리어 문구

---

## 1. 공통 원칙

### 1-1. 문구 길이
- 타이틀: 짧고 강하게
- 액션바: 지금 해야 할 행동 중심
- 업적/클리어 문구: 약간 더 길어도 됨

### 1-2. 문체 원칙
- 설명문보다 선언문이 좋음
- 보스의 성격이 드러나야 함
- 같은 테마 안에서는 어휘 톤을 통일할 것

### 1-3. 채널별 추천
- **타이틀**: 패턴 이름 / 페이즈 전환 / 발악 진입
- **서브타이틀**: 짧은 설명
- **액션바**: 카운트다운 / 개인 경고 / 즉시 행동 요구
- **클리어 문구**: 업적, 기록, 첫 처치 강조
