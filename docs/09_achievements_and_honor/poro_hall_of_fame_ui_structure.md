# 포로 서버 명예의 전당 / 기록 UI 구조 초안

## 문서 목적
이 문서는 극상위 보스 최초 클리어, 무사망 클리어, 시즌 기록 등을 보여주는
명예의 전당 UI 구조를 정리한 초안이다.

핵심 목표:
- 서버의 상징적인 기록을 한눈에 볼 수 있게 한다.
- 업적, 전승 보상, 보스 기록과 연결한다.
- 시즌성/영구성 기록을 구분한다.

---

## 1. 추천 탭 구조

- `서버 최초`
- `극상위 기록`
- `시즌 베스트`
- `무사망 기록`
- `전승 보유자`

---

## 2. 엔트리 표시 예시

- 보스 이름
- 클리어 파티원 이름
- 클리어 날짜
- 클리어 시간
- 무사망 여부
- 데카 0 생존 여부
- 대표 칭호/휘장

---

## 3. 시즌과 영구 기록 분리

### 영구 기록
- 서버 최초 클리어
- 최초 무사망
- 첫 전승 획득

### 시즌 기록
- 시즌 최단 클리어
- 시즌 최소 사망
- 시즌 최고 기여

---

## 4. 한 줄 요약

**명예의 전당은 서버 최초, 극상위 기록, 시즌 베스트, 전승 보유자 탭으로 나누고, 보스 기록과 대표 칭호/휘장을 함께 노출하는 구조가 가장 좋다.**
