# 포로 서버 업적 보상 / UI 구조 초안

## 문서 목적
이 문서는 업적 시스템이 실제로 유저에게 어떻게 보이고,
어떤 보상과 연결되는지 정리한 초안이다.

핵심 목표:
- 업적이 묻히지 않게 UI와 보상 구조를 분명히 한다.
- 칭호, 뱃지, 명예의 전당, 영지 전시와 연결한다.
- 보스 전승 보상과도 자연스럽게 이어지게 한다.

---

## 1. UI 추천 구조

### 업적 메뉴
- 카테고리별 탭
- 진행도 표시
- 완료 업적 강조
- 보상 미수령 표시

### 프로필 연결
- 대표 업적 1~3개 노출
- 칭호/뱃지 선택 장착

### 결과창 연동
- 보스 클리어 후 새 업적 달성 시 즉시 노출

---

## 2. 보상 구조

### 일반 업적
- 소형 칭호
- 프로필 배지
- 소량 골드/상징 재화
- 기록 문구

### 지역/보스 업적
- 지역 상징 칭호
- 영지 트로피
- 전용 휘장

### 극상위 업적
- 전승 칭호
- 후광/외형
- 명예의 전당 노출
- 계정 전승 유물

---

## 3. 한 줄 요약

**업적 시스템은 별도 메뉴, 프로필 대표 업적 노출, 보스 결과창 연동을 갖추고, 보상은 칭호·뱃지·영지 전시·명예의 전당 중심으로 연결하는 것이 가장 좋다.**
