# 포로 서버 업적 시스템 큰 틀 초안

## 문서 목적
이 문서는 포로 서버에서 어떤 방향으로 업적 시스템을 설계할지 정리한 초안이다.

핵심 목표:
- 업적이 단순 숫자 놀음이 아니라, 플레이 기록과 명예를 남기는 장치가 되게 한다.
- 전투 / 보스 / 탐험 / 생활 / 영지 / 특별 조건을 고르게 포괄한다.
- 칭호, 프로필 뱃지, 영지 전시, 명예의 전당과 연결되게 만든다.

---

## 1. 업적 카테고리 추천

### 전투 업적
- 첫 결전 클리어
- 무사망 클리어
- 데카 0 상태 생존 클리어
- 특정 보스 최초 클리어

### 탐험 업적
- 지역 핵심 구역 발견
- 숨겨진 루트 발견
- 테마별 유적/장치 발견

### 생활 업적
- 첫 제작 성공
- 만찬 첫 제작
- 영지 첫 수확
- 특정 생활 레벨 도달

### 영지 업적
- 영지 해금
- 시설 Lv3 달성
- 특정 테마 시설 설치
- 전시 트로피 배치

### 특별 업적
- 이벤트성 클리어
- 극상위 최초 클리어
- 특정 조건부 도전 성공

---

## 2. 업적 보상 방향

추천:
- 칭호
- 프로필 뱃지
- 기록 문구
- 영지 전시물
- 소량 상징 재화

비추천:
- 강한 전투 성능 보상
- 미달성 유저와 큰 격차를 만드는 보상

---

## 3. 한 줄 요약

**포로 서버 업적은 전투·보스·탐험·생활·영지·특별 도전으로 나누고, 성능 보상보다는 칭호·뱃지·기록·영지 전시 같은 명예형 보상과 연결하는 것이 가장 적합하다.**
