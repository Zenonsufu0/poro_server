# 포로 서버 전체 마스터 인덱스 초안

## 문서 목적
이 문서는 지금까지 정리한 포로 서버 문서를
**시스템별로 한 번에 찾아볼 수 있게 묶는 통합 인덱스** 초안이다.

핵심 목표:
- 어떤 문서가 어떤 시스템에 속하는지 빠르게 찾게 한다.
- Codex에 구현을 맡길 때 필요한 참조 문서를 빠르게 묶게 한다.
- 문서가 많아져도 구조를 잃지 않게 한다.

---

## 1. 보스 설계 핵심
- 보스 콘셉트 / 패턴 설계
- 결전 / 극상위 구조
- 체력 / 방어력 / 전투 시간
- 패턴 피해 체급
- 패턴 스크립트
- 패턴 seed/config
- 보상 / 상징 보상 / 전승 보상

## 2. 전투 / 클래스 / 성장
- 전사 / 암살자 / 마법사 스킬 수치
- 직업각인 / 공용각인
- 세트효과 / 룬 / 잠재 / 강화
- 상태이상 / 제어 / 표식 코드
- 전투 계산식 관련 기준

## 3. 생활 / 영지 / 소비 아이템
- 생활 자원
- 제작식
- 숙련도
- 영지 시설
- 음식 / 만찬 / 물약
- 생활 가격표
- 생활 적응 퀘스트

## 4. 스토리 / 퀘스트 / 지역
- 수도 허브 구조
- 지역 메인 퀘스트
- 필드 → 던전 → 보스 유도
- 자작령 / 백작령 / 후작령 구조
- NPC 역할
- 퀘스트 objective / reward / seed

## 5. 명예 / 업적 / UI
- 업적 마스터
- 칭호 / 뱃지 / 명예의 전당
- 결과창 문구
- 전승 보상
- 시즌 / 명예 시즌

## 6. 운영 / 웹 / 디코 봇
- 운영 대시보드
- KPI / API 예시 JSON
- 웹 운영 페이지
- 디코 봇 명령어 / 응답 포맷 / 슬래시 명세
- 플레이어 상세 조회 방향

## 7. 데이터 / seed / registry
- 장비 / 옵션 / 잠재
- 보스 패턴
- 스킬
- 퀘스트
- 업적
- NPC / 지역 / 마을
- T2 제작식 / 보스 재료 매핑

---

## 8. 한 줄 요약

**포로 서버 문서는 보스, 전투/성장, 생활/영지, 스토리/퀘스트, 명예/UI, 운영/조회, seed/config 일곱 묶음으로 보면 가장 찾기 쉽고 구현 연결도 좋다.**
