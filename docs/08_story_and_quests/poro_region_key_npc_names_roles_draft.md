# 포로 서버 지역별 핵심 NPC 이름 / 직책 초안

## 문서 목적
이 문서는 각 지역에서 반복적으로 등장하거나 퀘스트 허브 역할을 맡는
**핵심 NPC의 이름과 직책 감각**을 1차로 정리한 초안이다.

핵심 목표:
- 지역마다 기억에 남는 핵심 NPC 라인을 만든다.
- 퀘스트 허브, 조사관, 보급관, 기록관 등 공통 기능을 지역 테마에 맞게 구체화한다.
- 나중에 대사, 관계도, 세부 설정을 붙이기 쉬운 골격을 만든다.
