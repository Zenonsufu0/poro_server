# 포로 서버 스토리 / 퀘스트 시스템 큰 틀 초안

## 문서 목적
이 문서는 포로 서버의 스토리 / 퀘스트 시스템을 나중에 구체화하기 전에,
먼저 **전체 구조와 역할**을 잡아두기 위한 초안이다.

핵심 목표:
- 초반 스폰 직후 동선부터 지역별 스토리 퀘스트까지 큰 줄기를 만든다.
- 전투/탐험/보스 중심 서버 철학과 충돌하지 않게 퀘스트를 설계한다.
- 나중에 세부 텍스트, NPC 대사, 연출, 보상표를 붙일 수 있는 골격을 만든다.

중요:
- 이 문서는 아직 세부 대사/연출/분기문보다 **퀘스트 구조**를 먼저 정하는 문서다.
- 초안 단계에서는 "어느 시점에 어떤 퀘스트가 필요한가"를 고정하는 게 핵심이다.

---

## 1. 기본 철학

### 1-1. 퀘스트는 메인 길잡이이자 세계관 연결 장치다
포로 서버의 메인은 전투 / 보스 / 던전 / 탐험이다.
퀘스트는 이를 방해하는 숙제가 아니라,
- 초반 적응을 돕고
- 지역 진입 이유를 주고
- 보스와 던전을 스토리적으로 연결하고
- 유저가 세계를 기억하게 만드는 장치여야 한다.

### 1-2. 퀘스트는 크게 4축으로 나눈다
- **튜토리얼 / 적응 퀘스트**
- **메인 스토리 퀘스트**
- **지역 스토리 퀘스트**
- **반복/의뢰 퀘스트**

### 1-3. 스토리는 너무 길게 늘어뜨리지 않는다
포로 서버는 싱글 RPG가 아니다.
따라서
- 핵심 동선은 짧고 강하게
- 보스 전후 맥락은 분명하게
- 읽고 싶으면 더 볼 수 있는 방식
이 좋다.

---

## 2. 전체 퀘스트 구조

### 2-1. 스폰 직후
- 세계관 아주 짧은 인트로
- 직업 선택 또는 첫 전투 체험
- 수도 기초 동선 안내
- 첫 사냥 / 첫 NPC 상호작용 / 첫 보상
- 첫 지역 진출 전 최소 준비

### 2-2. 수도권 초반
- 제국 수도 소개
- 중앙 허브 NPC 소개
- 영지 / 제작 / 강화 / 창고 / 이동 핵심 UI 노출
- 첫 메인 퀘스트 시작

### 2-3. 지역 진출
각 지역은 아래 순서가 좋다.
1. 지역 진입 퀘스트
2. 지역 소개/위험 설명
3. 필드 사냥/탐험 퀘스트
4. 지역 스토리 퀘스트
5. 지역 던전/보스 연결
6. 결전 보스 연결

### 2-4. 후반
- 결전 보스 클리어 후 다음 지역 떡밥
- 여러 지역을 돌아보며 세계관 조립
- 극상위 보스는 메인 퀘스트 종결보다는 "도전형 전승 콘텐츠"로 분리

---

## 3. 스폰 직후 퀘스트의 역할

### 꼭 필요한 것
- 기본 이동
- 기본 공격
- 스킬 사용
- 첫 장비 지급
- 첫 강화 또는 첫 장비 교체
- 첫 지역 이동

### 너무 많이 넣지 말아야 하는 것
- 생활 시스템 전체 설명
- 복잡한 경제 설명
- 긴 세계관 낭독
- 강제 반복 의뢰

### 추천 감각
스폰 후 10~20분 안에
- 서버의 기본 조작을 익히고
- 수도를 이해하고
- 첫 필드로 나가게 만드는 게 좋다.

---

## 4. 메인 스토리 퀘스트 구조

### 추천 흐름
- 수도 이상 징후
- 각 지역의 문제 발견
- 지역별 결전 보스와 연결
- 배후 세력 / 공통 위협 암시
- 극상위 도전은 별도 전승 루트로 확장

### 메인 퀘스트의 역할
- 다음 지역을 왜 가는지 설명
- 결전 보스를 왜 잡는지 설명
- 지역별 세계관을 묶는 역할

---

## 5. 지역 스토리 퀘스트 구조

각 지역은 아래 정도가 적당하다.

### 지역당 추천
- 진입 퀘스트 1~2개
- 핵심 스토리 퀘스트 3~5개
- 던전/보스 연결 퀘스트 1~2개
- 반복 의뢰 2~4개

### 역할
- 지역의 분위기와 문제 제시
- 정예몹 / 던전 / 보스 연결
- 테마 자원 / 상태이상 / 전장 특징 학습

---

## 6. 반복 / 의뢰 퀘스트 구조

### 용도
- 골드 보조 수급
- 생활/필드 루프 보조
- 초반 동선 강화
- 일일/주간 루프 보조

### 주의
포로 서버는 반복 의뢰가 메인이 되면 안 된다.
따라서
- 보조 보상
- 짧은 완료 시간
- 메인 성장 루프 보완
정도가 적당하다.

---

## 7. 극상위와 퀘스트의 관계

극상위는 메인 스토리의 필수 종결로 두기보다,
**전승 / 진실 / 숨겨진 시련**에 가깝게 두는 편이 좋다.

즉:
- 메인 스토리 엔딩 이후 열리거나
- 특정 결전 보스들을 넘은 뒤 해금되거나
- 별도 도전 퀘스트 라인으로 분리

이 구조가 가장 자연스럽다.

---

## 8. 지금 당장 정해야 하는가?

### 지금 먼저 정하면 좋은 것
- 퀘스트 구조
- 스폰 직후 역할
- 메인/지역/반복 퀘스트 비중
- 각 지역별 퀘스트 개수 감각
- 극상위와 메인 스토리의 거리

### 나중에 정해도 되는 것
- NPC 이름 최종판
- 자세한 대사
- 컷신 연출
- 퀘스트 문장 텍스트
- 세부 보상표
- 분기 연출

즉:
**초안은 지금 같이 밀 수 있고, 세부 스토리 텍스트는 나중에 붙여도 충분하다.**

---

## 9. 한 줄 요약

**플레이어 초기 스폰 후 동선, 메인 스토리, 지역별 스토리 퀘스트, 반복 의뢰 구조는 지금 초안으로 먼저 밀어도 되고, 자세한 대사/연출/텍스트는 핵심 시스템 문서화를 더 끝낸 뒤 붙여도 충분하다.**
