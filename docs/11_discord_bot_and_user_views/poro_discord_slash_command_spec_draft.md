# 포로 서버 디스코드 슬래시 명령 최종 명세 초안

## 문서 목적
이 문서는 디코 봇 명령어를 실제 슬래시 명령 형태로 정리한 초안이다.

핵심 목표:
- command name, option, 응답 요약 구조를 미리 고정한다.
- 유저가 많이 쓰는 조회 명령 위주로 정리한다.
- 나중에 디코 봇 구현 시 그대로 옮기기 쉽게 만든다.

---

## 1. 명세 예시

### `/내정보`
- 옵션: 없음
- 응답: 직업, 무기, 총 강화, 대표 세트, 최근 클리어

### `/내장비`
- 옵션: `상세여부(optional)`
- 응답: 장비 부위 요약, 강화, 세트, 잠재 대표값

### `/보스기록`
- 옵션: `보스명(optional)`
- 응답: 최근 기록 또는 특정 보스 기록

### `/시세`
- 옵션: `아이템명(required)`
- 응답: 평균 시세, 거래량, 최근 변화

### `/내영지`
- 옵션: 없음
- 응답: 시설 목록, 레벨, 최근 수확, 대기 생산량

### `/명예의전당`
- 옵션: `카테고리(optional)`
- 응답: 서버 최초 / 극상위 / 시즌 기록 요약

---

## 2. 한 줄 요약

**디코 슬래시 명령은 /내정보, /내장비, /보스기록, /시세, /내영지, /명예의전당 같은 핵심 조회 명령부터 명세를 고정하는 것이 가장 실용적이다.**
