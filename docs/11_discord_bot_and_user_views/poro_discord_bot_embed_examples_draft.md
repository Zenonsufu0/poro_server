# 포로 서버 디스코드 봇 카드 응답 예시 초안

## 문서 목적
이 문서는 디코 봇이 실제로 유저에게 어떤 카드/임베드 형태로 응답하면 좋은지 예시를 정리한 초안이다.

핵심 목표:
- 유저가 많이 보는 명령어 위주로 보기 좋은 응답 예시를 만든다.
- 추후 디코 봇 embed 설계에 바로 참고할 수 있게 한다.

---

## 1. `/내장비` 예시

**현도의 장비 요약**
- 직업: 전사 / 대검
- 총 강화 합: 87
- 세트: 황성의 위광 4세트
- 대표 잠재: 핵심 태그 피해 +8%, 최대 체력 +6%
- 대표 각인: 파쇄의 권위 / 핵심 증폭 Lv5 / 회복 강화 Lv4

## 2. `/보스기록` 예시

**최근 보스 기록**
- 라그네스: 클리어 4회 / 최고 8분 42초
- 모르바인: 클리어 2회 / 최고 10분 11초
- 아그네르: 미클리어 / 최고 4페이즈 진입

## 3. `/내생활` 예시

**생활 요약**
- 채집 11 / 채광 9 / 벌목 7 / 낚시 8
- 요리 10 / 연금 8
- 영지 시설: 약초 재배지 Lv3 / 광물 추출장 Lv2
- 최근 수확: 설혼초 12, 정제 광석 8

## 4. `/시세 용암 결정핵` 예시

**시세 정보: 용암 결정핵**
- 최근 평균가: 1,980 골드
- 최근 거래량: 37
- 어제 대비: +4.2%
- 연결 제작: 남부 T2 장비

## 5. 한 줄 요약

**디코 봇 응답은 텍스트를 길게 늘어놓기보다 장비, 기록, 생활, 시세를 카드형 요약으로 보여주는 구조가 가장 좋다.**
