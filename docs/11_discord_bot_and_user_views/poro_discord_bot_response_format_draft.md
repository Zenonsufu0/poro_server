# 포로 서버 디스코드 봇 응답 포맷 초안

## 문서 목적
이 문서는 디코 봇이 유저에게 정보를 어떻게 보여줄지 응답 포맷 감각을 정리한 초안이다.

핵심 목표:
- 명령마다 너무 긴 응답이 나오지 않게 한다.
- 카드형 요약, 핵심 수치, 최근 기록 중심으로 보여준다.
- 나중에 embed 메시지 구조로 쉽게 옮길 수 있게 한다.

---

## 1. 내장비 응답 예시

### 표시 항목
- 직업 / 무기
- 장착 장비 요약
- 총 강화 합
- 대표 잠재 2~3개
- 세트효과 적용 상태

### 감각
한 화면에 핵심만 보이고,
세부는 `/내잠재`, `/내강화`로 나누는 게 좋다.

---

## 2. 보스기록 응답 예시

### 표시 항목
- 최근 클리어한 보스 5개
- 클리어 시간
- 최고 기록
- 무사망 여부
- 극상위 클리어 여부

---

## 3. 시세 응답 예시

### 표시 항목
- 아이템 이름
- 최근 평균 시세
- 최근 거래량
- 어제 대비 상승/하락
- 추천 매수/매도 감각(선택)

---

## 4. 한 줄 요약

**디코 봇 응답은 긴 통계표보다 장비 요약, 최근 기록, 핵심 수치 중심의 embed 카드형 구조로 가는 것이 가장 보기 좋다.**
