# 포로 서버 보스 패턴 설계 초안

이 문서는 **테마별 공용 패턴 방향**, **보스별 공용/전용 패턴 구분**, **개발자 코멘트**를 한 번에 정리한 마크다운 초안이다.

## 설계 원칙

- **공용 패턴**: 여러 보스가 공유하는 재사용 가능한 패턴 모듈
- **전용 패턴**: 해당 보스만의 상징 기믹 또는 전투 규칙
- **테마 스킨**: 같은 공용 패턴이라도 테마에 따라 이펙트·상태이상·우선순위를 달리 적용
- **개발자 코멘트**: 왜 이 패턴 구성을 넣었는지, 어떤 플레이 감각을 노렸는지

## 보스 계층별 전용 패턴 권장량

| 계층 | 공용 패턴 | 전용 패턴 |
|---|---:|---:|
| 필드보스 | 2~3 | 1 |
| 스토리 보스 | 3~4 | 1 |
| 최상위 보스 | 4~5 | 1~2 |
| 심층 보스 | 4~5 | 1~2 |
| 결전 보스 | 4~5 | 2 |
| 극상위 보스 | 4~5 | 3 |

# 수도권

## 테마 패턴 방향

### 공용 패턴 방향
- 전방 부채꼴 강타
- 직선 돌진 베기
- 선고 표식 후 폭발
- 구역 분할 장판
- 방패/결계 생성
- 일정 시간 후 중앙 심판 낙하

### 추천 전용 패턴 방향
- 판결 선고
- 재판 구획
- 봉인 해제

### 테마 코멘트
수도권은 제국·기사·재판·봉인 이미지가 강하므로, 정직하고 묵직한 정면 압박과 ‘심판’ 느낌의 공간 제어 패턴이 잘 어울린다.

## 필드보스

### 외성 파괴자 그론
- 위치: 황도 외성
- 공용 패턴: 전방 부채꼴 강타, 직선 돌진 베기
- 전용 패턴: 성문 파쇄
- 개발자 코멘트: 입문 필드보스라 복잡한 기믹보다 ‘정면으로 밀어버리는 위협’을 먼저 체험시키는 역할이다.

### 검은깃 집행조장
- 위치: 왕도 하층구역
- 공용 패턴: 짧은 연속 베기, 표식 후 도약 강타
- 전용 패턴: 암시장 포위망
- 개발자 코멘트: 골목 전투에 맞는 인간형 추격전을 체감시키기 위해 기동형 압박을 넣었다.

### 도로 습격단장 바르센
- 위치: 순례자 가도
- 공용 패턴: 직선 돌진, 부채꼴 휩쓸기
- 전용 패턴: 호송 차단
- 개발자 코멘트: 이동로 위협과 길목 통제라는 필드 특성을 보스 패턴에 반영했다.

### 망가진 수문장 헤르바크
- 위치: 폐성 감시초소
- 공용 패턴: 전방 강타, 원형 장판 폭발
- 전용 패턴: 붕괴 낙하물
- 개발자 코멘트: 무너진 방어시설이라는 공간성을 살리기 위해 구조물 붕괴 연출을 패턴으로 사용한다.

### 수로 포식자 바실
- 위치: 황도 지하수로
- 공용 패턴: 짧은 돌진, 원형 장판 폭발
- 전용 패턴: 오염 수로 범람
- 개발자 코멘트: 좁은 지하 공간에서 빠르게 회피 각을 잡아야 하는 긴장감을 만들려는 의도다.

## 스토리 보스

### 순찰대장 헤르몬
- 위치: 황도 외성 입구
- 공용 패턴: 전방 부채꼴 강타, 직선 돌진 베기, 방패/결계 생성
- 전용 패턴: 기사 반격 자세
- 개발자 코멘트: 첫 인스턴스 보스 역할이라 패턴은 읽기 쉬우면서도 타이밍 대응을 배우게 한다.

### 암거래 수장 벨타
- 위치: 왕도 하층구역 입구
- 공용 패턴: 표식 후 폭발, 짧은 연속 베기, 짧은 이동 후 사선 공격
- 전용 패턴: 독거래 폭약
- 개발자 코멘트: 하층구역 보스답게 정면 승부보다 속임수와 암습 압박이 핵심이다.

### 타락한 감시관 레오니스
- 위치: 폐성 감시초소 입구
- 공용 패턴: 직선 검기, 원형 장판 폭발, 중앙 심판 낙하
- 전용 패턴: 망령 소집
- 개발자 코멘트: 기사형 패턴에 타락과 망령 연출을 섞어 스토리 밀도를 높였다.

## 최상위 보스

### 제국 망각기사 알드렌
- 위치: 황도 지하수로 최심부
- 공용 패턴: 전방 부채꼴 강타, 직선 돌진 베기, 구역 분할 장판, 방패/결계 생성
- 전용 패턴: 왕실 검진
- 개발자 코멘트: 세트 입장선 보스라 정직하지만 학습 가능한 패턴으로 설계했다.

### 심연을 본 재판관
- 위치: 폐성 감시초소 붕괴 성소
- 공용 패턴: 선고 표식 후 폭발, 재판 구획, 중앙 심판 낙하, 방패/결계 생성
- 전용 패턴: 판결 역전
- 개발자 코멘트: 수도권 최상위 보스다운 상징성을 위해 재판·심판 계열 패턴을 집중 배치했다.

## 심층 보스

### 금쇄 파수관 드라벤트
- 위치: 봉인된 황도 지하성 입구
- 공용 패턴: 구역 분할 장판, 전방 부채꼴 강타, 직선 돌진 베기, 짧은 무력화 딜타임
- 전용 패턴: 쇠사슬 속박
- 개발자 코멘트: 후반부다운 이동 제한 압박을 주기 위해 속박 기믹을 추가했다.

### 망각기사 에르세인
- 위치: 망각의 재판회랑 입구
- 공용 패턴: 직선 돌진 베기, 방패/결계 생성, 원형 장판 폭발, 안전지대 찾기
- 전용 패턴: 기억 박탈
- 개발자 코멘트: 재판회랑 분위기에 맞춰 패턴 기억 자체를 흔드는 심리적 압박을 넣었다.

## 결전 보스

### 황혼의 대재판관 세르카인
- 위치: 재판회랑 심판대
- 공용 패턴: 전방 부채꼴 강타, 선고 표식 후 폭발, 구역 분할 장판, 안전지대 찾기, 짧은 무력화 딜타임
- 전용 패턴: 판결 선고, 재판 구획
- 개발자 코멘트: 수도권 테마를 대표해야 하므로 ‘심판’과 ‘법정’ 이미지를 가장 강하게 느끼게 만드는 구성을 택했다.

## 극상위 보스

### 제국 심연집행자 아우렐
- 위치: 봉인 해제 후 황도 심부
- 공용 패턴: 직선 돌진 베기, 중앙 심판 낙하, 구역 분할 장판, 방패/결계 생성, 안전지대 찾기
- 전용 패턴: 봉인 해제, 황도 계엄, 사형 집행 선언
- 개발자 코멘트: 제국이 끝까지 숨긴 최후의 무력을 체감하게 하는 최종 보스 구성을 의도했다.

# 서부 황야

## 테마 패턴 방향

### 공용 패턴 방향
- 돌진 추격
- 부채꼴 모래 파동
- 바닥 함몰
- 표식 후 도약 강타
- 시야를 흐리는 장막
- 이동 경로 차단 장판

### 추천 전용 패턴 방향
- 모래폭풍 추적
- 함몰지대
- 매몰 표식

### 테마 코멘트
서부 황야는 넓은 공간, 이동 제약, 추적과 생존이 핵심이라 발밑을 빼앗고 계속 따라붙는 패턴이 잘 어울린다.

## 필드보스

### 모래갈기 투르칸
- 위치: 바람마른 초원
- 공용 패턴: 돌진 추격, 부채꼴 모래 파동
- 전용 패턴: 사냥 본능
- 개발자 코멘트: 서부 입문 필드답게 직관적인 추격형 야수 보스로 설계했다.

### 협곡 약탈장 바르코
- 위치: 붉은모래 협곡
- 공용 패턴: 표식 후 도약 강타, 직선 돌진
- 전용 패턴: 협곡 낙하 유도
- 개발자 코멘트: 협곡의 좁은 지형을 체감시키기 위해 이동 실수 처벌을 넣었다.

### 백골 포식자 누르
- 위치: 뼈먼지 벌판
- 공용 패턴: 돌진 추격, 짧은 연속 강타
- 전용 패턴: 사체 포식 광폭화
- 개발자 코멘트: 포식자 보스인 만큼 체력이 줄수록 더 집요해지는 감각을 주고 싶었다.

### 철녹 감시장 마고르
- 위치: 잊힌 감시초소
- 공용 패턴: 직선 돌진, 부채꼴 휩쓸기
- 전용 패턴: 고철 파편 비산
- 개발자 코멘트: 폐초소 잔해를 무기로 쓰는 느낌을 주기 위해 금속 파편 연출을 넣었다.

### 매몰 수호수 자르카
- 위치: 모래먹는 폐허
- 공용 패턴: 원형 장판 폭발, 바닥 함몰
- 전용 패턴: 유적 각성
- 개발자 코멘트: 폐허와 지형 자체가 공격하는 인상을 만들기 위해 함몰과 각성 연출을 강조했다.

## 스토리 보스

### 붉은모래 집행관
- 위치: 붉은모래 협곡 입구
- 공용 패턴: 돌진 추격, 부채꼴 모래 파동, 표식 후 도약 강타
- 전용 패턴: 모래참수
- 개발자 코멘트: 서부 특유의 ‘쫓아오는 압박감’을 가장 먼저 명확히 느끼게 하려는 구성이다.

### 황야 검투사 마도르
- 위치: 잊힌 감시초소 입구
- 공용 패턴: 전방 강타, 직선 돌진, 짧은 무력화 딜타임
- 전용 패턴: 결투 선언
- 개발자 코멘트: 정직한 인간형 보스를 하나 두어 유저가 패턴 읽기 자체에 집중하도록 했다.

### 매몰 사제 자하르
- 위치: 모래먹는 폐허 입구
- 공용 패턴: 원형 장판 폭발, 이동 경로 차단 장판, 시야를 흐리는 장막
- 전용 패턴: 유적 저주
- 개발자 코멘트: 사제 보스답게 직접 때리기보다 공간을 저주로 제어하는 전투를 의도했다.

## 최상위 보스

### 황혼의 모래거인
- 위치: 모래먹는 폐허 심부
- 공용 패턴: 전방 부채꼴 강타, 원형 장판 폭발, 바닥 함몰, 짧은 무력화 딜타임
- 전용 패턴: 사막 거체 붕락
- 개발자 코멘트: 크고 느리지만 한 번 한 번이 무서운 황야의 생존 감각을 살리는 보스다.

### 피바람의 추적자 세란
- 위치: 뼈먼지 벌판 아래 혈풍 구덩이
- 공용 패턴: 돌진 추격, 표식 후 도약 강타, 이동 경로 차단 장판, 시야를 흐리는 장막
- 전용 패턴: 혈풍 표식
- 개발자 코멘트: 황야의 ‘추적자’ 이미지를 가장 직접적으로 체감하게 하는 단일 집중형 보스다.

## 심층 보스

### 무덤파수장 아크람
- 위치: 매몰왕의 협곡묘역 입구
- 공용 패턴: 직선 돌진, 원형 장판 폭발, 바닥 함몰, 안전지대 찾기
- 전용 패턴: 묘역 봉기
- 개발자 코멘트: 묘역 전체를 위험 구역으로 바꾸는 느낌으로 후반부 분위기를 살렸다.

### 붉은 추적귀 세자르
- 위치: 피바람의 모래심연 입구
- 공용 패턴: 돌진 추격, 표식 후 도약 강타, 시야를 흐리는 장막, 짧은 무력화 딜타임
- 전용 패턴: 심연 추적
- 개발자 코멘트: 심층 보스답게 타깃 집요함을 극대화해 계속 쫓겨다니는 느낌을 강화했다.

## 결전 보스

### 사막패왕 하자드
- 위치: 심연 모래제단
- 공용 패턴: 부채꼴 모래 파동, 바닥 함몰, 이동 경로 차단 장판, 안전지대 찾기, 짧은 무력화 딜타임
- 전용 패턴: 모래폭풍 추적, 함몰지대
- 개발자 코멘트: 황야가 유저를 삼키는 듯한 느낌이 핵심이라 시야·이동·지면 자체를 적으로 만들었다.

## 극상위 보스

### 모래폭군 네브카
- 위치: 고대 폭풍 관문
- 공용 패턴: 돌진 추격, 부채꼴 모래 파동, 바닥 함몰, 시야를 흐리는 장막, 안전지대 찾기
- 전용 패턴: 매몰 표식, 폭풍 장벽, 유사 안전지대
- 개발자 코멘트: 극상위 보스는 황야의 모든 요소를 총합해 ‘버티며 살아남는 전투’로 만들고 싶었다.

# 동부 숲

## 테마 패턴 방향

### 공용 패턴 방향
- 뿌리 속박 장판
- 분신 생성
- 은신 후 돌진
- 안개 구역 생성
- 투사체 유도탄
- 안전지대 찾기

### 추천 전용 패턴 방향
- 안개 분리
- 뿌리 제단
- 그림자 도약 표식

### 테마 코멘트

## 1. 공용 보스 패턴 모듈 표준화
- `PatternSlashCone`
- `PatternLineCharge`
- `PatternGroundAoE`
- `PatternSafeZoneCheck`
- `PatternSummonWave`
- `PatternLaserSweep`
- `PatternMarkThenExplode`
- `PatternSplitArena`

이걸 먼저 표준화하면 이후 보스 추가/수정이 훨씬 쉬워진다.

## 2. 보스 패턴 데이터 스키마 설계
예: `cast_time`, `telegraph_time`, `shape_type`, `damage_multiplier`, `debuff_type`, `cooldown`, `phase_condition`, `priority`

이걸 먼저 잡아두면 패턴을 코드에 하드코딩하지 않고 데이터로 조합할 수 있다.

## 3. 결전/극상위 보스 상세 패턴 명세 1차 작성
우선순위:
- 북부 설원 결전/극상위
- 남부 화산 결전/극상위
- 아르카논 결전/극상위

여기는 전용 기믹 비중이 높아서, 다음 단계에서 5~8패턴 수준으로 상세화하는 게 가장 효율적이다.
