# 포로 서버 보스 절대 체력 예시표 초안

## 문서 목적
이 문서는 이미 정리한 보스 체급표와 기준 파티 DPS 표를 바탕으로, 실제 구현에 쓸 수 있는 절대 체력 예시값을 제시하는 초안이다.

중요:
- 최종 확정치가 아니라 1차 예시값이다.
- 실제 알파/베타에서 가장 크게 조정될 가능성이 높은 문서다.
- 핵심은 결전/극상위 보스 체력이 어느 정도 범위에 있어야 하는지 출발점을 만드는 것이다.

---

## 1. 계산 전제

### 기준 파티
- 결전 기준: T2 초입 3인 파티
- 극상위 기준: T2 후반 3인 파티

### 임시 DPS 기준
- T2 초입 파티 총 DPS: 100,000
- T2 후반 파티 총 DPS: 152,000

### 목표 시간
- 결전: 8~13분
- 극상위: 12~18분

---

## 2. 결전 보스 절대 체력 예시

| 보스 | 추천 전투 시간 | 절대 체력 예시 |
|---|---|---:|
| 라그네스 | 8~10분 | 52,000,000 |
| 모르바인 | 9~11분 | 56,000,000 |
| 하자드 | 9~11분 | 56,000,000 |
| 세르카인 | 10~12분 | 61,000,000 |
| 카실레아 | 10~12분 | 61,000,000 |
| 아르케논 | 11~13분 | 67,000,000 |
| 아자켈 | 11~13분 | 68,000,000 |

### 메모
- 라그네스는 가장 기준점 역할
- 아르케논/아자켈은 후반 결전이므로 6천만 후반까지 올릴 수 있다

---

## 3. 극상위 보스 절대 체력 예시

| 보스 | 추천 전투 시간 | 절대 체력 예시 |
|---|---|---:|
| 아우렐 | 12~14분 | 118,000,000 |
| 네브카 | 12~14분 | 120,000,000 |
| 엘테론 | 13~15분 | 124,000,000 |
| 엘드하임 | 13~16분 | 130,000,000 |
| 아그네르 | 14~16분 | 136,000,000 |
| 세트라 | 14~17분 | 141,000,000 |
| 카르멘 | 15~18분 | 148,000,000 |

### 메모
- 극상위는 1.1억~1.5억 사이 감각으로 시작 가능
- 카르멘은 현 시점 대표 최종 체급

---

## 4. 발악 포함 여부
위 체력 예시는 발악 포함 전체 체력 감각이다.
즉 0% 진입 후 발악 구간도 전체 전투 시간 일부로 포함한다.

추천:
- 결전 발악 체력 비중: 총 체력의 4~6%
- 극상위 발악 체력 비중: 총 체력의 5~8%

---

## 5. 한 줄 요약

**1차 예시값 기준으로 결전 보스는 5,200만~6,800만, 극상위 보스는 1억 1,800만~1억 4,800만 정도의 체력 범위에서 시작하는 것이 가장 무난하다.**
