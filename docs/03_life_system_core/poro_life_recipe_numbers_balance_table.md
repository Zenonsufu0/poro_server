# 포로 서버 생활 제작식 상세 수치 보정표 초안

## 문서 목적
이 문서는 이미 정리한 생활 제작식 마스터 표를 바탕으로,
실제 구현과 테스트에 쓰기 위한 **재료 개수 / 제작 난이도 / 가치 감각**을 더 촘촘하게 정리한 초안이다.

핵심 목표:
- 어떤 제작식이 가벼워야 하고 어떤 제작식이 무거워야 하는지 구분한다.
- 일반 음식 / 만찬 / 물약 / 가공재의 체급 차이를 만든다.
- Codex가 recipe seed에 수량과 tier 값을 넣을 수 있게 한다.

---

## 1. 기본 규칙

### 가공재
- 가벼워야 함
- 중간재라 자주 돌릴 수 있어야 함

### 일반 음식
- 부담 없이 만들 수 있어야 함
- 자주 먹는 기본 소비품

### 만찬
- 확실히 무거워야 함
- 파티 준비용 고급 소비품

### 회복 물약
- 일반은 가볍게
- 희귀는 체감되게 무겁게

### 상태 대응 / 전투 보조 물약
- 일반 음식보다 조금 무겁거나 비슷
- 희귀 부산물 소량 섞여도 됨

- 희귀 재료는 1개 정도면 충분하다.
- 너무 무거우면 시장이 죽고, 너무 가벼우면 필수가 된다.

## 8. 제작 가치 체급 정리

### 낮은 체급
- 정제 약초
- 가공 목재
- 주괴
- 일반 회복 물약
- 기본 대응 물약

### 중간 체급
- 공명 분말
- 설혼 방한식
- 홍염 내열식
- 고급 회복 물약
- 전투 보조 물약

### 높은 체급
- 만찬 전반
- 희귀 회복 물약
- 테마 희귀 재료가 들어가는 특수 제작식

## 9. 한 줄 요약

**생활 제작식 수치는 가공재와 일반 소비품은 가볍게, 만찬과 희귀 회복 물약은 확실히 무겁게 설계하는 것이 좋으며, 일반 음식은 총 4~5개 재료, 만찬은 총 10~12개 재료 수준의 체감으로 시작하는 것이 가장 무난하다.**
