# 포로 서버 핵심 결정사항

## 서버 한 줄 방향
포로 서버는 오픈월드 탐험, 액션 전투, 월드 보스, 던전, 퀘스트를 중심으로 한 마인크래프트 RPG 서버다.

## 기술 기준
- 서버 버전: Paper 1.21.10
- Java: 21
- 개발 방식: Java + Gradle
- 운영 환경: Linux
- 데이터 저장: 초기 SQLite, 필요 시 확장

## 핵심 콘텐츠 우선순위
1. 기본 직업 시스템
2. 초반 전투 구조
3. 제국 수도 및 시작 동선
4. 월드 보스
5. 오픈월드 던전
6. 퀘스트 기반 성장 구조

## 전투 공용 원칙
- 쿨타임 기반 액션 전투
- 직업: 전사 / 암살자 / 마법사
- 공용 태그 6개:
  - 핵심
  - 연계
  - 이동
  - 정밀
  - 범위
  - 생존
- 스킬은 최대 2태그
- 태그 피해 증가는 합연산
- 피해 계산 틀:
  - 기본 피해 × (1+일반 피해 증가) × (1+태그 피해 증가) × (1+조건부 피해 증가) × 방어보정 × 치명보정 × (1-상대 피해감소)

## 상태 체계
- 버프
- 디버프
  - 상태이상: 출혈, 독, 화상, 냉기 침식
  - 제어: 둔화, 속박, 경직, 빙결
  - 표식류: 표적, 균열, 약점 노출, 낙인, 암영 표식

## 전투/성장 철학
- 레벨은 해금 중심
- 실질 전투력의 본체는 장비
- RNG는 고점에 개입, 플레이스타일 선택에는 개입하지 않음

## 레벨 구조
- 전투 레벨 최대 50
- 생활 레벨 분리
- 전투 레벨은 스킬/콘텐츠/T1→T2 해금 담당
- 레벨업 시 HP/공격력/방어력 소폭 증가
- 직업은 처음 선택 후 변경 불가
- 각인은 거점에서 변경 가능
