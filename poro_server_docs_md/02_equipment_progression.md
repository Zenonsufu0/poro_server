# 장비 성장 시스템

## 장비 성장 철학
- 장비 실물 드랍보다 흔적/전승 중심 성장
- 장비는 부위별 관리
- 세트, 등급, 잠재, 강화는 서로 다른 레이어로 관리

## 장비 정보 레이어
장비 1부위는 아래 정보를 독립적으로 가진다.
1. 베이스 장비
2. 강화 수치
3. 세트 정보
4. 아이템 등급
5. 잠재

예시:
- 베이스: T2 대검
- 강화: 12강
- 세트: 황성의 위광
- 등급: 유니크
- 잠재: 에픽

## 아이템 등급
등급 명칭은 모두 통일:
- 커먼
- 레어
- 에픽
- 유니크
- 레전더리

### 등급 역할
- 무기/방어구의 고정 수치와 옵션 수치 품질을 결정
- 동일 장비 기준 최하~최상 등급 총합 차이는 약 10% 내외

## 잠재
잠재 등급 명칭:
- 커먼
- 레어
- 에픽
- 유니크
- 레전더리

### 잠재 규칙
- T1 잠재 상한: 에픽
- T2는 에픽 시작
- T2 잠재 상한: 레전더리
- 잠재는 오직 큐브로만 변경/승급
- 유니크 흔적은 잠재가 아니라 아이템 등급 전용

## T1 / T2 구조
### T1
- 퀘스트 / 필드 / 스토리 중심
- 레벨 1~49 구간 메인 장비
- 최대 강화: 18강
- 잠재 상한: 에픽

### T2
- 전투 레벨 50부터 시작하는 최종 파밍 장비
- 보스 / 정예 / 세트 / 흔적 중심
- 일반 유저 종결: 18강
- 그 이상은 열심히 하는 유저 도전 구간

## T1 → T2 승급 전승
- 전승 단위: 부위별
- 전승 조건: 해당 T1 장비 18강
- 전승 비용: 무료 (스토리 승급 처리)
- 전승 결과: T2 장비 5강 시작
- 기준 성능: T2 5강 = T1 18강 동급
- 전승 후 T1 장비는 소멸

## 세트 흔적
- 부위별 제작
- 제작 재료: 보스 전용 재료 + 골드
- 세트 흔적을 장비에 전승하면 해당 부위가 세트 장비로 변환

## 세트 변환
- 한 번 해금한 세트끼리만 가능
- 해금은 업적 시스템으로 관리
- 변환 비용: 골드만
- 같은 부위끼리만 변환 가능

## 등급 전승
- 높은 등급 흔적을 획득하면 언제든 전승 가능
- 비용: 골드 + 특수 재료
- 유지:
  - 강화 수치
  - 세트 정보
  - 잠재
- 변경:
  - 아이템 등급만 변경

## 상위권 고점
- 아이템 등급
- 잠재
- 공용 각인 레벨
- 무기 21강 이상

## 강화 규칙
- 파괴 없음
- 하락 없음
- 성공 / 실패만 존재
- 최대 강화 25강
- 일반 유저 종결선은 T2 18강
- 19강 이상은 열심히 하는 유저 / 상위권 구간
