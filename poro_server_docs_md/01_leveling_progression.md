# 전투 레벨 구조

## 기본 방향
- 전투 레벨 최대 50
- 생활 레벨은 별도
- 전투 레벨은 전투력의 본체가 아니라 해금 축
- 장비, 세트, 잠재, 각인 운용이 전투력의 본체

## 레벨 구간 의미
### 1~10
- 튜토리얼
- 기본 스킬과 전투 적응
- 시작 지역, 기초 퀘스트

### 11~20
- 무기 기본 루프 완성
- 첫 던전 / 정예 / 미니보스
- Lv.20에 무기/직업 메커니즘 완성

### 21~30
- 각인 선택 구간
- Lv.25에 직업 각인 선택
- 플레이스타일 분기 시작
- T1 중반 장비와 중반 던전

### 31~40
- 흔적/전승/잠재 활용 시작
- 상위 지역과 보스 적응
- T1 후반 세팅 마무리

### 41~50
- 최종 지역 진입
- T2 선행 퀘스트
- Lv.50에 T2 해금 및 선택한 각인 최종 완성 효과 개방

## 각인 해금 구조
- Lv.20: 무기/직업 메커니즘 완성
- Lv.25: 직업 각인 선택
- Lv.50: 선택한 각인의 최종 완성 효과 해금

## 생활 레벨 방향
- 영지 생산
- 약초 재배
- 물약 제작
- 보조 성장
- 전투 필수는 아니지만 있으면 확실히 이득
- 다음 프로젝트에도 재사용 가능하도록 모듈형 설계
