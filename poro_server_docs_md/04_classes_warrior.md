# 전사 무기 정리

## 공통
- 대검 / 태도 / 창
- 공용 태그:
  - 핵심
  - 연계
  - 이동
  - 정밀
  - 범위
  - 생존

---

## 대검
### 기본 패시브
- 철갑

### 각인
- 파쇄
- 중갑

### 핵심 구조
- 파쇄: 균열을 쌓고 소모해 폭발
- 중갑: 철갑 스택을 오래 유지하거나 광전으로 전환

### 주요 스킬
- 철벽자세
- 광전의 외침
- 파쇄참
- 올려베기
- 분쇄 파동
- 패왕격
- 천붕 낙하

## 태도
### 기본 패시브
- 검세

### 각인
- 거합
- 잔심

### 분류
- 검세
- 발도
- 잔심
- 심안
- 일섬

### 주요 이름
- 초도
- 거합참
- 절영참
- 일섬
- 심안참
- 무심

### 핵심 구조
- 거합: 납도/발도 중심 찰나 폭발
- 잔심: 연계 유지와 저장 일섬 활용

---

## 창
### 기본 패시브
- 압박

### 각인
- 파진
- 진압

### 분류
- 압박
- 쇄입
- 제압
- 봉세
- 결단

### 핵심 구조
- 압박 최대 6
- 압박 6 달성 시 붕괴
- 붕괴 중 결단 스킬 강화
- 결단 여러 개 채용 가능
- 단, 붕괴 1회당 결단 사용은 1회
- 결단 사용 시 붕괴 종료

### 각인 차이
- 파진: 단일 붕괴/관통흔 중심
- 진압: 압박 전이/진각 중심

- 횡월
- 유창
- 봉쇄세
- 응수
- 관철
- 반월
