# Wuthering Waves chat reference batch 02

이 ZIP은 사용자가 채팅에 붙여넣은 최근 무기 레퍼런스 이미지를 묶은 것입니다.

포함 범위:
- 장리 Blazing Brilliance 관련 이미지 후보
- 저해상도 4각도 주황 검 후보
- 파란 검 디테일 보조
- 빨강/은색 기계형 무기 보조
- 나무 샤프트 창/봉 헤드 보조
- 업로드 원본 순서 보존 폴더

주의:
- 일부 이미지는 채팅 첨부 순서 기반으로 자동 분류했으므로, `00_raw_uploaded_sequence`를 최종 기준으로 확인하세요.
- 검은 프레임/빈 프레임 후보는 `99_unsorted/uncertain_support`에 넣었습니다.
