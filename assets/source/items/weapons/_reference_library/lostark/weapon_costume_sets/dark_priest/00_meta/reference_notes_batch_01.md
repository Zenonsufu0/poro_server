# Lost Ark Dark Priest - Batch 01

## Summary
Dark Priest is valuable for shadow priest, heretic, corrupted holy gear, assassin weapons, dark mage weapons, and high-rarity dark costume language.

## Strong Uses

### Assassin Weapons
- assassin_weapon_grid: A
- Blade: use for 02_katana selected candidate.
- Reaper: use for 04_shortsword selected candidate.
- Demonic: use for 09_arks / corrupted artifact / dark transformation support.

### Warrior Weapons
- warrior_weapon_grid: A-
- Berserker greatsword is useful for 01_greatsword dark variant.
- Warlord lance/shield is useful for 03_spear support.
- Holyknight sword is useful for dark holy sword guard language.
- Destroyer hammer is future/support only.

### Mage Weapons
- mage_weapon_grid: A-
- Staff and long staff are useful for 07_staff selected/support.
- Arcanist frame is useful for 08_orbs selected/support.
- Bard instrument is support/future only.

### Martial Artist Weapons
- martial_artist_weapon_grid: A-
- Qigong orb/core is useful for 09_arks or 08_orbs support.
- Spear and gauntlet are support candidates.

### Hunter Weapons
- hunter_weapon_grid: B+
- Useful mainly for dark gun and blaster support.
- Not a primary server direction unless Arcanon firearms expand.

## Costume Direction
- black/silver priest armor: shadow priest, heretic, dark paladin.
- burgundy/gold priest armor: brand, blood, judgment, ritual noble.
- female black armor: assassin/dark priest costume reference.
- male armor closeups: chest, pauldron, mask, silver ornament density.

## Poro Server Fit
- Hongyeom Brand style
- Shadow Mark style
- corrupted holy order
- dark priest dungeon NPC
- world boss cosmetic reward

## Avoid
- exact recreation
- exact symbols/logos/emblems
- over-thin spikes in Blockbench
- making all dark gear pure black with no readable metal separation
