# Legendary Eternity - Mage & Assassin Batch

## Summary
This batch is valuable for staff/orb/katana/dagger references. It is less useful for armor density than the warrior batch, but very useful for magical and assassin weapon language.

## Strong Candidates

### 07_staff
- mage_staff_grid: A-
- mage_specialist_longstaff_grid: A-
- Use for elegant staff proportions, crown/star tips, holy/royal mage weapon variants.

### 08_orbs
- mage_arcanist_deck_grid: A
- Use as magical deck/orb-frame direction.
- Supports the rule: Orb = outer magical frame / card / ring / symbolic structure.
- mage_bard_harp_grid: B+
- Use as ritual frame support, not primary weapon template.

### 02_katana
- assassin_blade_sword_grid: A
- Important because katana references were previously weak.
- Use for elegant curved blade / assassin sword silhouette.

### 04_shortsword
- assassin_reaper_dagger_grid: A
- Use for curved dagger, short assassin weapon, shadow/precision weapon variants.

### 09_arks / Shadow Artifact
- assassin_demonic_weapon_grid: A-
- Use as demonic/void/unstable artifact support.
- Good for corrupted weapon language and dark magic variants.

### Future Scythe
- assassin_souleater_scythe_grid: A-
- Use for future scythe, boss weapon, death/shadow cosmetic.

## Costume Direction
- mage costumes: holy/elf/priestess/ritual mage skin support.
- assassin costumes: oriental assassin, dark ritual, demonic shadow, death priestess support.

## Avoid
- exact costume recreation
- over-thin weapon geometry in Blockbench
- copying exact Lost Ark class identity
- making all mage weapons too decorative without readable silhouette
