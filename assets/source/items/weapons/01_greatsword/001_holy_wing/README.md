# 001_holy_wing

This asset is built using the template/module workflow.

## Workflow

1. Collect external visual references in `refs/web_refs`.
2. Write `reference_synthesis.md`.
3. Build or approve:
   - base_greatsword_01
   - angel_wing_medium_01
   - guard_gem_celestial_01
   - pommel_gem_round_01
4. Assemble final holy_wing_greatsword.
5. Apply final texture.
6. Export Minecraft Java item model.
