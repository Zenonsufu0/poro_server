# poro_flame_guardian_sword_25d — 2.5D Build Plan

## 레퍼런스 요약

원본 이미지(`poro_flame_guardian_sword_cutout_full.png`)에서 확인된 구조:

- **검신**: 길고 얇은 날. 중앙부가 가장 넓고 팁으로 갈수록 좁아짐. 양쪽 가장자리에 불꽃 톱니 돌출부 3~4개. 날 끝에 뾰족한 불꽃 스파이크.
- **가드**: 상단에 위쪽으로 휘는 곡선 날개(좌우 대칭). 중앙에 큰 다이아몬드형 보석. 가드 하단에 짧은 아래쪽 스파이크 2개.
- **손잡이**: 어두운 브라운/블랙 랩 그립. 중간 길이.
- **폼멜**: 다이아몬드/마름모 형태.
- **색상 무드**: 딥 크림슨(어두운 주황/갈색) 베이스 + 브라이트 오렌지 불꽃 강조.

## 2.5D 설계 원칙

| 부위 | 깊이(Z) | 근거 |
|------|---------|------|
| 검신 코어 | 1u | 평면 판 — 불꽃 디테일은 텍스처로 |
| 검신 톱니 (edge strips) | 1u | 실루엣 돌출만 geometry |
| 검신 팁 | 1u | 팁도 평면 |
| 가드 허브 / 하단 | 2u | 약간의 두께로 3D 존재감 |
| 가드 날개 팔 | 2–3u | 날개에 두께 필요 (손에 쥘 때 입체감) |
| 중앙 보석 | 2u | 입체 큐브 + 보석 텍스처 |
| 손잡이 | 1u | 단순 평면 |
| 폼멜 | 2u | 작은 입체 블록 |

## 텍스처 전략

| 영역 | 텍스처 내용 |
|------|-------------|
| 검신 코어 정면 | 중앙 다크 크림슨 → 가장자리 밝은 오렌지 그라데이션. 중앙 세로 라인 강조. |
| 검신 코어 측면 | 매우 얇아 1px — 다크 오렌지 단색 |
| 검신 엣지 톱니 | 밝은 오렌지/노란 불꽃 픽셀 |
| 가드 날개 | 다크 브라운 베이스 + 오렌지 엣지 강조 |
| 보석 | 밝은 오렌지 코어, 다이아몬드 패턴, 흰/노랑 하이라이트 |
| 손잡이 | 다크 브라운/블랙 랩, 밴드는 다크 골드 |
| 폼멜 | 다크 브라운 + 오렌지 센터 |

**금지**: 색상 모자이크 배열, 필로우 쉐이딩, 랜덤 노이즈 픽셀

## 빌드 순서 (bb-build-blockout 시)

1. 손잡이 + 폼멜 먼저 (Y 기준점 확정)
2. 가드 하단 코어 + 스파이크
3. 가드 허브 + 보석
4. 가드 날개 팔 (위쪽 스위핑 — 각 3큐브 계단식)
5. 검신 코어 (가장 긴 파트)
6. 검신 엣지 톱니 (4쌍)
7. 검신 팁 + 스파이크

## 품질 게이트 체크리스트

- [ ] 전면 뷰에서 불꽃 대검 실루엣 즉시 인식 가능
- [ ] 검신이 납작한 직사각형 막대가 아님 (엣지 톱니로 실루엣 깨짐)
- [ ] 가드 날개가 위쪽으로 스위핑 (레퍼런스 방향 일치)
- [ ] 보석이 중앙에 입체감 있게 돌출
- [ ] 손잡이와 폼멜이 분리되어 읽힘
- [ ] Z 깊이: 검신=1u, 가드=2-3u, 손잡이=1u
- [ ] 총 큐브 수 ≤ 35
